Pętla for

Pętla for to pętla iteracyjna, która pozwala na wykonanie określonej liczby powtórzeń.

Pętla for to jedna z podstawowych pętli w języku C#, która służy do iteracji przez elementy tablicy lub kolekcji. Składnia pętli for wygląda następująco: 

								
for (inicjalizacja; warunek; iteracja) {

	// kod, który ma zostać wykonany w pętli
}
Wyjaśnienie

W powyższej składni, "inicjalizacja" to wyrażenie, które jest wykonywane na początku pętli, "warunek" to wyrażenie logiczne, które jest sprawdzane przed każdą iteracją, a "iteracja" to wyrażenie, które jest wykonywane na końcu każdej iteracji.

Przykład użycia pętli for

								
using System;

class Program
{
		static void Main(string[] args)
		{
				for (int i = 0; i < 5; i++)
				{
						Console.WriteLine(i);
				}
		}
}

Wyjaśnienie

Zmienna 'i' jest inicjalizowana na wartość 0, a następnie jest zwiększana o 1 za każdym razem, gdy kod w pętli jest wykonywany. Pętla będzie się powtarzać tak długo, jak długo i będzie mniejsze niż 10. W rezultacie na ekranie zostaną wyświetlone liczby od 0 do 9.

Pętla while

Pętla while jest pętlą warunkową, która wykonuje swój blok kodu tak długo, jak długo podany warunek jest spełniony.

Pętla while to kolejna podstawowa pętla w języku C#, która służy do wykonywania bloku kodu, dopóki warunek jest spełniony. Składnia pętli while wygląda następująco: 

								
while (warunek) {

	// kod, który ma zostać wykonany w pętli
}
Wyjaśnienie

W powyższej składni, "warunek" to wyrażenie logiczne, które jest sprawdzane przed każdą iteracją, a blok kodu jest wykonywany, dopóki warunek jest prawdziwy.

Przykład użycia pętli while

								
using System;

class Program
{
		static void Main(string[] args)
		{
				int i = 0;

				while (i < 10) {

						Console.WriteLine(i);

						i++;
				}
		}
}

Note

warunek w pętli while jest sprawdzany przed wykonaniem kodu w pętli. Jeśli warunek jest fałszywy już na początku, kod w pętli nie zostanie wykonany ani razu.

Pętla do while

Pętla do while jest podobna do pętli while, z tą różnicą, że kod w jej bloku wykonuje się przynajmniej raz, niezależnie od tego, czy warunek jest spełniony, czy nie. 

								
do {

	// kod, który ma zostać wykonany w pętli

} while (warunek);

W przeciwieństwie do pętli while, która sprawdza warunek na początku, pętla do…while sprawdza warunek na końcu pętli. 

								
using System

class Program
{
		int i = 0;

		do {

		Console.WriteLine(i);

		i++;

		} while (i < 10);
}

Zagnieżdzanie Pętli for

Zagnieżdżenie pętli polega na umieszczeniu jednej pętli wewnątrz drugiej. Dzięki temu możliwe jest wykonanie pewnego kodu wielokrotnie w różnych kombinacjach. 

								
for (int i = 0; i < 3; i++) {

	for (int j = 0; j < 3; j++) {

		Console.WriteLine($"i = {i}, j = {j}");
	}
}

W rezultacie na ekranie zostaną wyświetlone następujące linie: 

								
i = 0,  j = 0

i = 0,  j = 1

i = 0,  j = 2

i = 1,  j = 0

i = 1,  j = 1

i = 1,  j = 2

i = 2,  j = 0

i = 2,  j = 1

i = 2,  j = 2

Zagnieżdzenie Pętli while i do while

Pętle while i do while mogą być również zagnieżdżane w ten sam sposób, co pętle for. 

								
int i = 0;

while (i < 3) {

	int j = 0;

	while (j < 3) {

		Console.WriteLine($"i = {i}, j = {j}");

		j++;
	}

	i++;
}

Przykład zagnieżdżenia pętli do while: 

								
int i = 0;

do {

	int j = 0;

	do {

		Console.WriteLine($"i = {i}, j = {j}");

		j++;

	} while (j < 3);

	i++;

} while (i < 3);

Polecenie break

Słowo kluczowe break służy do przerwania działania pętli lub switch. Gdy program wykona polecenie break, natychmiast przerywa on działanie pętli lub switch i przechodzi do następnego polecenia poza pętlą lub switch.

Uwaga

Możesz również użyć słowa kluczowego break w pętli while lub do while, oraz w strukturze switch. W przypadku zagnieżdżonych pętli, polecenie break przerywa działanie. 

								
for (int i = 0; i < 10; i++) {

	if (i == 5) {

		break;

	}

	Console.WriteLine(i);
}
Wyjaśnienie

W powyższym przykładzie pętla for wypisze na ekranie liczby od 0 do 4, a następnie zostanie przerwana przez polecenie break, gdy i osiągnie wartość 5.

Polecenie continue

Słowo kluczowe continue służy do przerwania działania bieżącej iteracji pętli i przejścia do następnej. Gdy program wykona polecenie continue, natychmiast przerywa on wykonywanie pozostałego kodu w pętli i przechodzi do następnej iteracji. 

								
for (int i = 0; i < 10; i++) {

	if (i % 2 == 1) {

		continue;

	}

	Console.WriteLine(i);
}
Wyjaśnienie

W powyższym przykładzie pętla for wypisze na ekranie liczby parzyste od 0 do 8, ponieważ dla liczb nieparzystych polecenie continue przerywa wykonywanie pozostałego kodu w pętli i przechodzi do następnej iteracji.
