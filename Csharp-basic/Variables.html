Zmienne

Zmienna w języku C# to pojemnik, który może przechowywać wartość. Zmienne są używane do przechowywania wielu różnych danych które są wykorzystywane w programie. Każda zmienna musi mieć określony typ, który określa rodzaj danych, jakie może przechowywać. Na przykład zmienna typu int (liczba całkowita) może przechowywać liczby całkowite, zmienne typu string ciąg znaków.

Zmienne mogą być używane w programie do przechowywania różnych danych, takich jak liczby, ciągi znaków, obiekty i wiele innych. 

								
int x; // deklaracja zmiennej x typu int

string name; // deklaracja zmiennej name typu string (ciąg znaków)

Definiowanie zmiennych w C#

Definiowanie zmiennych to proces tworzenia zmiennych w programie. W języku C#, aby utworzyć zmienną, należy najpierw zadeklarować ją, określając jej typ i nazwę. Następnie można przypisać jej początkową wartość, jeśli to konieczne.

Jeśli zadeklarujesz zmienną typu int o nazwie "y", ale nie przypiszesz jej żadnej wartości początkowej, to jej domyślna wartość będzie równa 0. 

								
int y;   //Domyślna wartość jest równa 0

int x = 8;  //Deklaracja zmiennej x oraz domyślne przypisanie wartości 8

string name = "Jake"  //deklaracja zmiennej name oraz przypisanie jej wartości "Jake"

Możesz również zadeklarować kilka zmiennych jednocześnie, odzielając je przcinkami, możesz wybrać odpowiedni typ danych dla swoich potrzeb i użyć go do zadeklarowania zmiennej. Więcej informacji o typach danych w języku C# można znaleźć karcie "Typy Dancyh". 

								
int a, b, c;  //deklaracja trzech zmiennych typu int o nazwach a, b i c

Inicjowanie zmiennych w C#

Inicjowanie zmiennych to proces przypisywania początkowej wartości. przy użyciu operatora przypisania = możesz inicjować zmienne. Inicjowanie zmiennych jest ważne, ponieważ pozwala uniknąć błędów w kodzie i zapewnia, że zmiennebędą miały odpowiednie wartości początkowe dla swoich typów.

Inicjowanie bez przypisania wartości

												
int liczba; // inicjacja zmiennej typu int bez przypisania wartości
Wyjaśnienie

W tym przypadku zmienna "liczba" zostanie utworzona, ale nie będzie miała przypisanej żadnej wartości. Przed zainicjowaniem zmiennej, jej odwołanie spowoduje błąd kompilacji.

Inicjowanie z przypisaniem wartości

												
int liczba = 10; // inicjacja zmiennej typu int z przypisaniem wartości 10
Wyjaśnienie

W tym przypadku typ zmiennej zostanie automatycznie określony przez kompilator na podstawie przypisanej wartości, w tym przypadku będzie to typ "int".

Inicjowanie z użyciem "var"

												
var liczba = 10; // inicjacja zmiennej typu int z wykorzystaniem słowa kluczowego "var"
Wyjaśnienie

W tym przypadku zmienna "liczba" zostanie utworzona i od razu będzie miała przypisaną wartość 10.

Inicjacja wielu zmiennych jednocześnie

												
int liczba1 = 10, liczba2 = 20; // inicjacja dwóch zmiennych typu int z przypisanymi wartościami 10 i 20
Wyjaśnienie

W tym przypadku możemy zainicjować wiele zmiennych jednocześnie, oddzielając je przecinkami.

Akceptowanie danych

od użytkownika

W języku C# istnieje kilka sposobów na akceptowanie danych od użytkownika. Jednym z najprostszych sposobów jest użycie metody Console.ReadLine() Metoda ta pozwala odczytać ciąg znaków, który zostanie wprowadzony przez użytkownika za pomocą klawiatury.

Pobieranie danych z konsoli za pomocą metody ReadLine(): 

								
 Console.WriteLine("Proszę podać swoje imię:");

 string name = Console.ReadLine();

//output: Ciąg znaków wpisany przez użytkownika

Metoda Console.ReadLine() zwraca ciąg znaków, więc jeśli chcesz zapisać dane wejściowe użytkownika do zmiennej typu innego niż string. Możesz użyć klasy Convert 

								
 Console.WriteLine("ile ma lat twój komputer:")

 int wiek_komputera = Convert.ToInt32(Console.WriteLine());

//output: Liczba wpisana przez użytkownika w konsoli

Innym sposobem na akceptowanie danych od użytkownika jest użycie metody Parse lub TryParse 

								
 Console.WriteLine("Podaj swój wiek:");

 string wpisWieku;

 int wiek = int.Parse(wpisWieku);

//output: Liczba wpisana przez użytkownika w konsoli
