Niejawna Konwersja Typów

Konwersja niejawna, zwana również konwersją automatyczną, polega na przekształceniu jednego typu na inny bez konieczności użycia operatora rzutowania lub metody konwersji. Jest to możliwe, gdy jeden typ jest podtypem drugiego.

Przykład konwersji niejawnej: 

								
int x = 5;

long y = x; // konwersja niejawna z int na long

Niektóre typy danych nie mogą być skonwertowane na inne typy niejawnie, w takim przypadku wymagane jest użycie operatora rzutowania lub metody konwersji. 

								
float a = 5.5f;

int b = a; //konwersja niejawna z float na int. utrata precyzji

Jawna Konwersja Typów

Konwersja jawna, zwana również konwersją rzutowaniem, polega na przekształceniu jednego typu na inny przy użyciu operatora rzutowania lub metody konwersji. Jest to potrzebne, gdy konwersja niejawna jest niemożliwa lub gdy chcemy zmienić typ danych, niezależnie od tego, czy jest to bezpieczne czy nie.

Przykład konwersji jawnej za pomocą operatora rzutowania: 

								
double a = 5.5;

int b = (int)a; // jawna konwersja z double na int

Przykład konwersji jawnej za pomocą metody konwersji: 

								
string str = "5";

int number = Convert.ToInt32(str); // jawna konwersja z string na int

Metody Konwersji Typów

W języku C# istnieje kilka metod konwersji typów, które pozwalają na bezpieczne przekształcanie jednego typu na inny. Oto kilka przykładów:

Metoda Convert.ToInt32() 

								
string str = "5";

int number = Convert.ToInt32(str);

Metoda Convert.ToDouble() 

								
string str = "5.5";

double number = Convert.ToDouble(str);

Metoda Convert.ToBoolean() 

								
string str = "true";

bool b = Convert.ToBoolean(str);

Metoda Convert.ToChar() 

								
string str = "a";

char c = Convert.ToChar(str);

Metoda Convert.ToDecimal() 

								
string str = "5.5";

decimal number = Convert.ToDecimal(str);

Metoda Convert.ToDateTime() 

								
string str = "2022-01-01";

DateTime date = Convert.ToDateTime(str);

Metoda Convert.ToByte() 

								
int i = 255;

byte b2 = Convert.ToByte(i);
Uwaga

Te metody są przydatne gdy chcesz przekształcić typ danych na inny ale nie jesteś pewien czy jest to bezpieczne.
