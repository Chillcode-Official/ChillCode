Komentarze

W języku C#, komentarze to fragmenty kodu źródłowego, które nie są kompilowane i nie wpływają na działanie programu. Służą one do dodawania informacji dla innych programistów lub do dokumentowania kodu.

Komentarze w C# to między innymi komentarz Jednolinijkowy oraz Wielolinijkowy

jednolinijkowe 

				
// To jest komentarz jednolinijkowy w C#
int x = 5; // zmienna x zawiera wartość 5

Wieloliniowe

				
/*
To jest komentarz wielolinijkowy w C#. Może obejmować wiele linii tekstu.
Poniżej znajduje się przykładowy kod, który nie zostanie wykonany, ponieważ jest zakomentowany.
*/

// int y = 10; // ten kod jest zakomentowany i nie zostanie wykonany

int x = 8;
Ważne

Dobre praktyki programistyczne zalecają stosowanie komentarzy w celu dokumentowania kodu, ponieważ ułatwia to późniejsze jego zrozumienie i utrzymanie.
