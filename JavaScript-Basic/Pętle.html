Pętla (for) JavaScript

Pętla for jest jednym z głównych typów pętli w języku JavaScript, która pozwala wykonywać określoną czynność wiele razy, aż do momentu, gdy warunek nie jest już spełniony. Składa się z trzech części: inicjalizacji, warunku i aktualizacji.

Składnia pętli for wygląda następująco: 

                
for (inicjalizacja; warunek; aktualizacja) {

  // kod, który będzie wykonywany w pętli
}
  1. Inicjalizacja: jest wykonywana tylko raz na początku pętli i jest używana do zainicjalizowania zmiennej liczącej.
  2. Warunek: jest sprawdzany przed każdym wykonaniem kodu w pętli. Jeśli jest prawdziwy, kod w pętli jest wykonywany, w przeciwnym razie pętla jest zakończona.
  3. Aktualizacja: jest wykonywana po każdym wykonaniu kodu w pętli i jest używana do aktualizacji zmiennej liczącej.

Oto przykład pętli for w języku JavaScript, która wypisuje liczby od 1 do 10. W tym przykładzie, zmienna i jest zainicjalizowana jako 1, warunek sprawdzania i <= 10 jest wykonywany przed każdym wykonaniem kodu w pętli, a zmienna i jest aktualizowana o 1 po każdym wykonaniu kodu w pętli. 

                
for (let i = 1; i <= 4; i++) {

  console.log(i);
}

//output:

1
2
3
4

Pętla (while) JavaScript

Pętla while jest używana do powtarzania określonego kodu, dopóki warunek jest spełniony. Składa się tylko z dwóch części: warunku i kodu.

Składnia pętli while wygląda następująco: 

                
while (warunek) {

  // kod, który będzie wykonywany w pętli
}
Uwaga

Warunek jest sprawdzany przed każdym wykonaniem kodu w pętli. Jeśli jest prawdziwy, kod w pętli jest wykonywany, w przeciwnym razie pętla jest zakończona.

Oto przykład pętli while w języku JavaScript, która wypisuje liczby od 1 do 10. W tym przykładzie, zmienna i jest zainicjalizowana jako 1, a warunek sprawdzania i <= 10 jest wykonywany przed każdym wykonaniem kodu w pętli. Zmienna i jest aktualizowana o 1 po każdym wykonaniu kodu w pętli. 

                
let i = 1;

while (i <= 3) {

  console.log(i);
  i++;
}

//output:

1
2
3

Pętla (do..while) JavaScript

Pętla while jest używana do powtarzania określonego kodu, dopóki warunek jest spełniony. Składa się tylko z dwóch części: warunku i kodu.

Składnia pętli do..while wygląda następująco: 

                
do {

  // kod, który będzie wykonywany w pętli

} while (warunek);
Uwaga

Warunek jest sprawdzany po każdym wykonaniu kodu w pętli. Jeśli jest prawdziwy, kod w pętli jest wykonywany ponownie, w przeciwnym razie pętla jest zakończona.

Oto przykład pętli do...while w języku JavaScript, która wypisuje liczby od 1 do 10. W tym przykładzie, zmienna i jest zainicjalizowana jako 1, a kod w pętli jest wykonywany co najmniej raz. Warunek sprawdzania i <= 10 jest wykonywany po każdym wykonaniu kodu w pętli i, jeśli jest prawdziwy, kod jest wykonywany ponownie. Zmienna i jest aktualizowana o 1 po każdym wykonaniu kodu w pętli. 

                
let i = 1;

do {

  console.log(i);
  i++;

} while (i <= 5);

//output:

1
2
3
4
5
