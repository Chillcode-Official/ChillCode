Najważniejsze funkcję matematyczne

dostępne w języku JavaScript wraz z przykładami użycia:

Math.abs(x) - zwraca wartość bezwzględną liczby x.

Math.ceil(x) - zwraca najmniejszą liczbę całkowitą większą lub równą x.

					
Math.abs(-5); // 5
					
				
 					
Math.ceil(4.2); // 5

Math.floor(x) - zwraca największą liczbę całkowitą mniejszą lub równą x.

Math.max(a,b) - zwraca największą wartość spośród argumentów a, b..

					
Math.floor(4.9); // 4
					
				
					
Math.max(1, 2, 3, 4); // 4

Math.min(a,b) - zwraca najmniejszą wartość spośród argumentów a, b..

Math.pow(x, y) - zwraca wartość x podniesione do potęgi y.

					
Math.min(1, 2, 3, 4); // 1
					
				
					
Math.pow(2, 3); // 8

Math.round(x) - zwraca najbliższą liczbę całkowitą dla liczby x.

Math.sqrt(x) - zwraca pierwiastek kwadratowy liczby x.

					
Math.round(4.5); // 5
 				
				
					
Math.sqrt(16); // 4

Math.random() - zwraca losową liczbę z przedziału [0, 1).

Math.sin(x) - zwraca sinus kąta x wyrażonego w radianach.

					
Math.random(); // 0.123456789
 				
				
					
Math.sin(Math.PI/2); // 1

Math.cos(x) - zwraca kosinus kąta x wyrażonego w radianach.

Math.log(x) - zwraca logarytm naturalny liczby x.

					
Math.cos(Math.PI); // -1
 				
				
					
Math.log(10); // 2.302585092994046
Note

Te przykłady pokazują, jak użyć funkcji matematycznych w języku JavaScript do wykonywania różnych obliczeń.
