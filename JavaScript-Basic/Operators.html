Operatory Arytmetyczne

W języku JavaScript istnieją operatory arytmetyczne, które pozwalają na wykonywanie podstawowych działań matematycznych, takich jak dodawanie, odejmowanie, mnożenie i dzielenie. Oto kilka przykładów operatorów arytmetycznych:

+ dodaje dwie wartości.

- odejmuje jedną wartość od drugiej.

										
let x = 5 + 10;
15 
										
let x = 5 - 10;
-5

* mnoży dwie wartości.

/ dzieli jedną wartość przez drugą.

										
let x = 5 * 10;
50 
										
let x = 50 / 10;
5

% reszta z dzielenia dwóch wartości.

** Potęguje wartość zmiennej.

										
let x = 5 % 2;
1 
										
let x = 2 ** 3;
8

++ zwiększa wartość zmiennej o jeden.

-- zmniejsza wartość zmiennej o jeden.

										
let x = 5;
x++;
6 
										
let x = 5;
x--;
4
