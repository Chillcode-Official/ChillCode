Metody

Metody w JavaScript to funkcje, które są dostępne dla różnych typów danych, takich jak tablice, stringi czy obiekty. Każdy typ danych ma swoje unikalne metody, które umożliwiają operowanie na nich w wygodny sposób. Dzięki wykorzystaniu metod w JavaScript można wykonywać wiele różnych operacji na danych, takich jak dodawanie, usuwanie, sortowanie, filtrowanie, mapowanie, redukowanie i wiele innych.

Metody są bardzo pomocne, ponieważ pozwalają na skracanie kodu i wykonywanie złożonych operacji w sposób bardziej zwięzły i czytelny. Oto przykład prostej metody: 

						
// Definicja obiektu
let person = {

  firstName: "John",
  lastName: "Doe",
  age: 30,
  fullName: function() {
    return this.firstName + " " + this.lastName;
  }
};


// Wywołanie metody fullName() na obiekcie person i zapisanie wyniku do zmiennej name
let name = person.fullName();


// Wyświetlenie wyniku
console.log(name);
Wyjaśnienie

W tym przykładzie użyliśmy metody fullName(), która zwraca pełne imię i nazwisko osoby przechowywanej w obiekcie person. Metoda ta jest wywoływana na obiekcie person i zwraca wynik, który jest zapisywany do zmiennej name. Następnie wynik jest wyświetlany w konsoli za pomocą metody console.log().
