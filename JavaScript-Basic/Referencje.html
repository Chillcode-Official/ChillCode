Referencje

Referencje w języku JavaScript to wskaźniki do obiektów, które są przechowywane w pamięci. Obiekt jest tworzony przez referencję, co oznacza, że zmienna przechowująca obiekt jest w rzeczywistości wskaźnikiem do obiektu w pamięci, a nie jego kopią.

Można przypisać jedną zmienną do innej zmiennej i obie zmienne będą wskazywać na ten sam obiekt. W poniższym przykładzie przypisujemy obiekty do zmiennych. 

                
let obj1 = { name: "John" };
let obj2 = obj1;

console.log(obj1 === obj2); // true

Jeśli zmodyfikujemy obiekt poprzez jedną zmienną, to zmiany będą widoczne dla drugiej zmiennej

Aby uniknąć takiej sytuacji, można utworzyć kopię obiektu, zamiast przypisywać jego referencję.

                    
obj1.name = "Jane";

console.log(obj2.name); // "Jane"

                    
                
                    
let obj1 = { name: "John" };
let obj2 = Object.assign({}, obj1);

console.log(obj1 === obj2); // false
Uwaga

Podsumowując, referencje w języku JavaScript są istotne, ponieważ pozwalają na tworzenie obiektów i przypisywanie ich do zmiennych, ale jednocześnie należy pamiętać, że referencje do tego samego obiektu wskazują na ten sam obiekt i zmiany w jednym z nich będą widoczne dla wszystkich pozostałych.

Użycie Referencji

Oto kilka różnych przykładów użycia referencji w języku JavaScript:

Przekazywanie obiektów jako argumentów do funkcji

Modyfikacja obiektów w tablicach

                    
function changePerson(person) {

  person.name = "Bob";
}

let myPerson = { name: "Alice" };
changePerson(myPerson);

console.log(myPerson.name); // "Bob"

                    
                
                    
let people = [
  { name: "Alice", age: 30 },
  { name: "Bob", age: 40 },
  { name: "Charlie", age: 50 }
];

let person = people[0];
person.name = "Dave";

console.log(people[0].name); // "Dave"

Dostęp do właściwości obiektu 

                
let myPerson = { name: "Alice", age: 30 };
let name = myPerson.name;

console.log(name); // "Alice"

Przypisanie zmiennej wartości referencji do innego obiektu

Tworzenie nowych obiektów na podstawie innych obiektów

                    
let myPerson = { name: "Alice", age: 30 };
let anotherPerson = myPerson;
anotherPerson.name = "Bob";

console.log(myPerson.name); // "Bob"

                    
                
                    
let myPerson = { name: "Alice", age: 30 };
let anotherPerson = Object.assign({}, myPerson);
anotherPerson.name = "Bob";

console.log(myPerson.name); // "Alice"
console.log(anotherPerson.name); // "Bob"
Uwaga

W tych przykładach widać, jak referencje pozwalają na tworzenie, modyfikowanie i przekazywanie obiektów w języku JavaScript. Jednocześnie, ze względu na to, że referencje są wskaźnikami do obiektów, ważne jest, aby unikać nieoczekiwanych modyfikacji obiektów poprzez referencje i stosować kopie obiektów, gdy jest to konieczne.
