Zmienne

Zmienna w języku JavaScript to nazwane miejsce w pamięci komputera, które służy do przechowywania danych. Zmienne są używane do przechowywania informacji, takich jak liczby, teksty, obiekty i inne typy danych.

Aby utworzyć zmienną w języku JavaScript, należy użyć słowa kluczowego var , let lub const sekwencji (równa się) i wartości, którą chcemy przypisać do zmiennej. 

					
var x = 10;
let y = 20;
const z = 30;

if (true) {
  var x = 100; // ponowne przypisanie wartości dla zmiennej var x
  let y = 200; // zmienna y jest zmienną blokową i nie będzie wpływać na wartość zmiennej y poza blokiem if
  const z = 300; // zmienna z jest zmienną blokową i nie będzie wpływać na wartość zmiennej z poza blokiem if
  console.log(x, y, z); // wyświetli 100, 200, 300
}

console.log(x, y, z); // wyświetli 100, 20, 30

x = 1000;
y = 2000;
// z = 3000; - wywoła błąd, ponieważ zmienna const z nie może być ponownie przypisana

console.log(x, y, z); // wyświetli 1000, 2000, 30
