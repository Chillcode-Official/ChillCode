Operatory Arytmetyczne JavaScript

W języku JavaScript istnieją operatory arytmetyczne, które pozwalają na wykonywanie podstawowych działań matematycznych, takich jak dodawanie, odejmowanie, mnożenie i dzielenie. Oto kilka przykładów operatorów arytmetycznych:

+ dodaje dwie wartości.

- odejmuje jedną wartość od drugiej.

                    
let x = 5 + 10; // x = 15
                    
                
                    
let x = 5 - 10; // x = -5

* mnoży dwie wartości.

/ dzieli jedną wartość przez drugą.

                    
let x = 5 * 10; // x = 50
                    
                
                    
let x = 50 / 10; // x = 5

% reszta z dzielenia dwóch wartości.

** Potęguje wartość zmiennej.

                    
let x = 5 % 2; // x = 1
                    
                
                    
let x = 2 ** 3; // x = 8

++ zwiększa wartość zmiennej o jeden.

-- zmniejsza wartość zmiennej o jeden.

                    
let x = 5;
x++; // x = 6
                    
                
                    
let x = 5;
x--; // x = 4

Operatory Porównania

Operatory porównania w języku JavaScript pozwalają na porównywanie dwóch wartości i zwracanie wartości logicznej true lub false. Oto kilka przykładów operatorów porównania:

== Zwraca true, jeśli wartości są równe.

=== true gdy wartości i typy są równe.

                    
let x = 5 + 10; // x = 15
                    
                
                    
let x = 5 - 10; // x = -5

!= Zwraca true, jeśli wartości nie są równe.

!== Zwraca true, jeśli wartości lub typy nie są równe.

                    
let x = 5 * 10; // x = 50
                    
                
                    
let x = 50 / 10; // x = 5

> Zwraca true, jeśli pierwsza wartość jest większa od drugiej.

< Zwraca true, jeśli pierwsza wartość jest mniejsza od drugiej.

                    
let x = 5 % 2; // x = 1
                    
                
                    
let x = 2 ** 3; // x = 8

>= Zwraca true, jeśli pierwsza wartość jest większa lub równa drugiej.

<= Zwraca true, jeśli pierwsza wartość jest mniejsza lub równa drugiej.

                    
let x = 5;
x++; // x = 6
                    
                
                    
let x = 5;
x--; // x = 4

Operatory Logiczne

Operatory logiczne w języku JavaScript pozwalają na kombinowanie wartości logicznych i zwracanie wartości logicznej true lub false. Oto kilka przykładów operatorów logicznych:

&& Jeśli oba operandy są równe true, wynik jest równy true. W przeciwnym razie wynik jest równy false.

|| Jeśli co najmniej jeden z operandów jest równy true, wynik jest równy true. W przeciwnym razie wynik jest równy false.

                    
let x = true;
let y = false;

console.log(x && y); // false
                    
                
                    
let x = true;
let y = false;

console.log(x || y); // true

! Zmienia wartość logiczną operanda. true staje się false, a false staje się true.

                        
let x = true;

console.log(!x); // false
Uwaga

Operatory logiczne są często używane w warunkach, instrukcjach if i switch, a także w pętlach, aby kontrolować, kiedy wykonywać konkretne czynności.

Operatory Przypisania

Operatory przypisania w języku JavaScript pozwalają na przypisanie wartości do zmiennych. W języku JavaScript istnieją również inne operatory przypisania, takie jak <<=, >>=, >>>=, &=, ^= i |=, ale są one mniej często używane. O to kilka przykładów operatorów przypisania:

= Przypisuje wartość do zmiennej.

+= Dodaje wartość do zmiennej i zapisuje wynik z powrotem do zmiennej.

                    

let x = 10;

                    
                
                    
let x = 10;
x += 5;
console.log(x); // 15

-= Odejmuje wartość od zmiennej i zapisuje wynik z powrotem do zmiennej.

*= Mnoży wartość zmiennej i zapisuje wynik z powrotem do zmiennej.

                    
let x = 10;
x -= 5;
console.log(x); // 5
                    
                
                    
let x = 10;
x *= 5;
console.log(x); // 50

/= Dzieli wartość zmiennej i zapisuje wynik z powrotem do zmiennej.

%= Oblicza resztę z dzielenia wartości zmiennej i zapisuje wynik z powrotem do zmiennej.

                    
let x = 10;
x /= 5;
console.log(x); // 2
                    
                
                    
let x = 10;
x %= 3;
console.log(x); // 1

Operator Warunkowy

Operator warunkowy (ternary operator) w języku JavaScript pozwala na wykonanie jednej z dwóch operacji na podstawie wartości logicznej. Składa się z trzech części: warunku, który jest sprawdzany, wartości, która jest zwracana, jeśli warunek jest prawdziwy, i wartości, która jest zwracana, jeśli warunek jest fałszywy. Oto przykład operatora warunkowego:

W poniższym przykładzie operator warunkowy ? : sprawdza, czy x jest większe od 5. Jeśli tak, wynikiem jest "x jest większe od 5", a jeśli nie, wynikiem jest "x jest mniejsze lub równe 5" 

                
let x = 10;
let result = (x > 5) ? "x jest większe od 5" : "x jest mniejsze lub równe 5";
console.log(result); // x jest większe od 5
Uwaga

Operator warunkowy jest często używany jako alternatywa dla instrukcji if-else, kiedy jedna linia jest wystarczająca do wykonania określonej czynności. Jest to szczególnie przydatne w sytuacjach, gdy chcesz skrócić długą instrukcję if-else do jednej linii kodu.

Inne Operatory JavaScript

typeof i instanceof to operatory, które służą do określania typu obiektu lub zmiennej w języku JavaScript.

typeof to operator binarny, który zwraca ciąg tekstowy reprezentujący typ wartości, którą przedstawia. Oto kilka przykładów użycia typeof: 

                        
console.log(typeof "hello"); // string
console.log(typeof 123); // number
console.log(typeof true); // boolean
console.log(typeof {}); // object
console.log(typeof []); // object
console.log(typeof null); // object
console.log(typeof undefined); // undefined
console.log(typeof function() {}); // function

instanceof to operator binarny, który służy do określania, czy obiekt jest instancją danej klasy lub funkcji konstruktora. Oto przykład użycia instanceof: 

                        
let array = [];
console.log(array instanceof Array); // true
console.log(array instanceof Object); // true


let str = "hello";
console.log(str instanceof String); // false
console.log(str instanceof Object); // false
Wyjasnienie

W powyższym przykładzie, operator instanceof sprawdza, czy obiekt array jest instancją klasy Array lub Object. Analogicznie dla ciągu tekstowego str. Wartość false jest zwracana, ponieważ ciągi tekstowe w języku JavaScript są typem prostszym i nie są instancjami klas String lub Object.
