Tablice

Tablica w języku JavaScript to obiekt służący do przechowywania wielu wartości (elementów w jednej zmiennej. Każdy element może być różnego typu (np. liczba, ciąg znaków, obiekt, inna tablica, itd.).

Tablica jest tworzona za pomocą nawiasów kwadratowych, w których elementy są oddzielone przecinkami: 

								
var fruits = ['banana', 'apple', 'orange'];

Można dostać się do poszczególnych elementów tablicy za pomocą indeksu , który jest liczbą całkowitą wskazującą pozycję elementu w tablicy (indeksowanie jest zero-based, czyli pierwszy element ma indeks 0, drugi indeks 1, itd.). 

								
console.log(fruits[0]);
banana

Można również zmienić wartość danego elementu w tablicy: 

								
fruits[1] = 'pear';
console.log(fruits[1]);
pear
Note

Tablice są bardzo ważnym elementem języka JavaScript i są szeroko wykorzystywane do przechowywania danych, przetwarzania ich i wykonywania różnych operacji na tych danych.

Inne funkcje Tablic

Tablice w języku JavaScript są dynamiczne, co oznacza, że można zmieniać ich długość w trakcie działania programu. Można dodawać nowe elementy do tablicy lub usuwać już istniejące.

Można również iterować przez elementy tablicy, np. za pomocą pętli for lub foreach 

										
for (var i = 0; i < fruits.length; i++) {

	console.log(fruits[i]);
}
  • banana
  • pear
  • orange
										
fruits.forEach(function(fruit) {

	console.log(fruit);
});
  • banana
  • pear
  • orange

Tablica jest również obiektem , co oznacza, że można do niej dodawać właściwości : 

								
fruits.color = 'yellow';

console.log(fruits.color); // Output: "yellow"
yellow
Note

Ważne jest, aby pamiętać, że dodawanie właściwości do tablicy nie jest dobrym praktycznie rozwiązaniem i powinno się tego unikać.

Rodzaje Tablic

W języku JavaScript istnieją cztery rodzaje tablic, a mianowicie:

  • Tablice jednowymiarowe
  • Tablice wielowymiarowe
  • Tablice rzadkie
  • Tablice asocjacyjne

Tablice jednowymiarowe

Tablice jednowymiarowe to najprostszy rodzaj tablicy, który składa się z jednego wiersza wartości. W przypadku tablicy jednowymiarowej każdy element tablicy posiada indeks liczbowy, zaczynając od zera. 

								
let fruits = ['apple', 'banana', 'cherry'];

Tablice wielowymiarowe

Tablice wielowymiarowe to tablice, które składają się z wielu wierszy wartości. W przypadku tablicy wielowymiarowej, każdy wiersz posiada indeks liczbowy, podobnie jak w przypadku tablicy jednowymiarowej. Tablica wielowymiarowa może mieć dowolną ilość wierszy i kolumn. 

								
let matrix = [
	[1, 2, 3],
	[4, 5, 6],
	[7, 8, 9]
];

Tablice rzadkie

Tablice rzadkie to tablice, które zawierają brakujące wartości lub mają wartości ustawione na null, lub undefined. W przypadku tablicy rzadkiej, indeksy liczbowe nie muszą być ustawione kolejno i mogą być pominięte. 

								
let sparseArray = [];
sparseArray[0] = 'apple';
sparseArray[2] = 'banana';

Tablice asocjacyjne

Tablice asocjacyjne to tablice, w których elementy są indeksowane przez ciągi znaków (nie tylko przez liczby całkowite, jak w przypadku standardowych tablic). Tablice asocjacyjne w języku JavaScript są tworzone za pomocą obiektów. 

								
let person = { name: 'John', age: 30, city: 'New York' };

Właściwości I Metody Tablicy

Tablice w języku JavaScript mają następujące właściwości i metody, które umożliwiają przetwarzanie i manipulowanie ich elementami. Oto kilka przykładów użycia metod i właściwości tablic w języku JavaScript:

length właściwość, która zwraca liczbę elementów w tablicy.

push() metoda, która dodaje element na koniec tablicy.

										
let fruits = ['apple', 'banana', 'cherry'];

console.log(fruits.length);
3 
										
let fruits = ['apple', 'banana', 'cherry'];
fruits.push('orange');
console.log(fruits);
[ 'apple', 'banana', 'cherry', 'orange' ]

pop() metoda, która usuwa ostatni element z tablicy.

shift() metoda, która usuwa pierwszy element z tablicy.

										
let fruits = ['apple', 'banana', 'cherry'];
let lastFruit = fruits.pop();
console.log(fruits);
console.log(lastFruit);
[ 'apple', 'banana' ]
cherry 
										
let fruits = ['apple', 'banana', 'cherry'];
let firstFruit = fruits.shift();
console.log(fruits);
console.log(firstFruit);
[ 'banana', 'cherry' ]
apple

unshift() metoda, która dodaje element na początek tablicy.

splice() metoda, która pozwala na dodawanie lub usuwanie elementów z określonego miejsca w tablicy.

										
let fruits = ['apple', 'banana', 'cherry'];
fruits.unshift('orange');
console.log(fruits);
[ 'orange', 'apple', 'banana', 'cherry' ] 
										
let fruits = ['apple', 'banana', 'cherry'];
fruits.splice(1, 1, 'orange');
console.log(fruits);
[ 'apple', 'orange', 'cherry' ]

slice() metoda, która tworzy nową tablicę składającą się z części oryginalnej tablicy.

indexOf() metoda, która zwraca indeks pierwszego wystąpienia określonego elementu w tablicy.

										
let fruits = ['apple', 'banana', 'cherry'];
let newFruits = fruits.slice(1, 2);
console.log(fruits);
console.log(newFruits);
[ 'apple', 'banana', 'cherry' ]
[ 'banana' ] 
										
let fruits = ['apple', 'banana', 'cherry'];
let index = fruits.indexOf('banana');
console.log(index);
1

join() metoda, która łączy wszystkie elementy tablicy w jeden łańcuch znaków.

reverse() metoda, która odwraca kolejność elementów w tablicy.

										
let fruits = ['apple', 'banana', 'cherry'];

let joinedFruits = fruits.join(', ');

console.log(joinedFruits);
apple, banana, cherry 
										
let fruits = ['apple', 'banana', 'cherry'];

fruits.reverse();

console.log(fruits);
[ 'cherry', 'banana', 'apple' ]

sort() metoda, która sortuje elementy tablicy.

forEach() metoda, która umożliwia wykonanie funkcji na każdym elemencie tablicy.

										
let fruits = ['apple', 'banana', 'cherry'];
fruits.sort();

console.log(fruits);
[ 'apple', 'banana', 'cherry' ] 
										
let fruits = ['apple', 'banana', 'cherry'];
fruits.forEach(function(fruit) {
console.log(fruit);
});
  • apple
  • banana
  • cherry
Note

Te przykłady pokazują tylko niektóre z dostępnych metod i właściwości tablic w języku JavaScript. Są one bardzo użyteczne w wielu różnych sytuacjach i pozwalają na wykonywanie różnych operacji na tablicach.
