Obsługa Zdarzeń

Obsługa zdarzeń jest bardzo prosta i polega na dodaniu do elementu HTML funkcji, która będzie wykonywana w momencie zaistnienia określonego zdarzenia. W celu dodania funkcji obsługującej zdarzenie można skorzystać z wyżej wspomnianych trzech sposobów.

Niektóre ze zdarzeń, które można obsłużyć w języku JavaScript.

  • click - Kliknięcie przycisku
  • change - Zmiana wartości pola tekstowego
  • keypress - Naciśnięcie klawisza na klawiaturze
  • mousemove - Przesuwanie kursora myszki
  • load - Załadowanie strony
  • resize - Zmiana rozmiaru okna przeglądarki

Zmiana rozmiaru okna

								
<div id="myDiv">Zmień rozmiar okna przeglądarki</div>

<script>

  const div = document.querySelector('#myDiv');

  window.addEventListener('resize', () => {
    div.textContent = `Nowy rozmiar okna: ${window.innerWidth} x ${window.innerHeight}`;


</script>

Note

Pamiętaj, że obsługa zdarzeń jest jednym z najważniejszych aspektów tworzenia interaktywnych stron internetowych i jest bardzo pomocna w codziennym życiu programisty. Dlatego warto poświęcić czas na zgłębienie tego tematu.
