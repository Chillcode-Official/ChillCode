Funkcje JavaScript

W języku JavaScript funkcje są niezwykle ważnymi i potężnymi konstrukcjami językowymi, które pozwalają na organizację i reutilizację kodu. Funkcje są również ważne dla programowania funkcyjnego i pozwalają na przekazywanie i zwracanie wartości.

Aby utworzyć funkcję w języku JavaScript, można użyć następującej składni: 

                
function nazwaFunkcji(argumenty) {

    // Ciało funkcji
    // ...
    return wartość;
}

Aby wywołać funkcję, należy podać jej nazwę i, opcjonalnie, argumenty: 

                
nazwaFunkcji(argumenty);

W języku JavaScript istnieją również funkcje strzałkowe (arrow functions), które są krótszym i bardziej elastycznym sposobem tworzenia funkcji: 

                
const nazwaFunkcji = (argumenty) => {

    // Ciało funkcji
    // ...
    return wartość;
}
Uwaga

Funkcje w języku JavaScript są również przekazywane jako argumenty innych funkcji i mogą być zwracane jako wartość z innej funkcji. To pozwala na budowanie skomplikowanych aplikacji i programowanie funkcyjne.

Uwaga

Jednym z ważnych konceptów związanych z funkcjami jest ich zmienność. Funkcje w języku JavaScript są zmiennymi i mogą być przypisane do innych zmiennych, a nawet przekazywane jako argumenty innych funkcji. Funkcje są także w stanie zwracać inne funkcje jako wartości zwrotne.

Funkcja Anonimowa (JavaScript)

Funkcja anonimowa to funkcja, która nie posiada nazwy. Są one często używane w języku JavaScript jako argumenty przekazywane do innych funkcji lub jako funkcje zwrotne. Funkcje anonimowe są również często używane do tworzenia funkcji, które są tylko raz używane w aplikacji.

Oto przykład funkcji anonimowej w języku JavaScript: 

                
// Funkcja anonimowa jako argument przekazywany do funkcji setTimeout
setTimeout(function() {

  console.log("Czas minął!");

}, 1000);


//output: Czas minął!

Można również przypisać funkcję anonimową do zmiennej i użyć jej jako funkcji zwrotnej: 

                
let przywitajSię = function(imię) {

  console.log("Witaj, " + imię);
};

przywitajSię("Jan");


//output: "Witaj, Jan"
Uwaga

Funkcje anonimowe są często używane w języku JavaScript do tworzenia funkcji, które są używane tylko raz, a następnie są usuwane. Jest to przydatne, ponieważ pozwala uniknąć zanieczyszczenia globalnego zasięgu zmiennych.

Wszystkie właściwości Funkcji

W języku JavaScript funkcja jest obiektem, więc posiada kilka właściwości i metod. Oto kilka najważniejszych właściwości funkcji:

Właściwość prototype jest wbudowanym obiektem, który jest dostępny dla wszystkich funkcji. Służy do tworzenia dziedziczenia między obiektami. 

                
function Person(imię, nazwisko) {
  this.imię = imię;
  this.nazwisko = nazwisko;
}

Person.prototype.przedstawSię = function() {
  console.log("Cześć, jestem " + this.imię + " " + this.nazwisko);
};

var jan = new Person("Jan", "Kowalski");
jan.przedstawSię(); // Wyświetli: "Cześć, jestem Jan Kowalski"

length zwraca liczbę argumentów, jakie funkcja oczekuje na wejściu.

name zwraca nazwę funkcji.

                    
function sum(a, b) {
  return a + b;
}

console.log(sum.length); // Wyświetli: 2
                    
                
                    
function sum(a, b) {
  return a + b;
}

console.log(sum.name); // Wyświetli: "sum"

arguments jest obiektem zawierającym argumenty, które zostały przekazane do funkcji.

call() pozwala wywołać funkcję i przekazać do niej listę argumentów.

                    
function sum() {
  var result = 0;
  for (var i = 0; i < arguments.length; i++) {
    result += arguments[i];
  }
  return result;
}

console.log(sum(1, 2, 3, 4)); // Wyświetli: 10
                    
                
                    
function przywitajSię(imię) {

  console.log("Witaj, " + imię);
}

var imiona = ["Jan", "Anna", "Piotr"];
for (var i = 0; i < imiona.length; i++) {
  przywitajSię.call(null, imiona[i]);
}

Właściwość apply() pozwala wywołać funkcję i przekazać do niej tablicę argumentów. 

                
function sum() {
  var result = 0;
  for (var i = 0; i < arguments.length; i++) {
    result += arguments[i];
  }
  return result;
}

var liczby = [1, 2, 3, 4];
console.log(sum.apply(null, liczby)); // Wyświetli: 10

Właściwość bind() tworzy nową funkcję, która jest połączona z bieżącą funkcją i przypisana do określonego kontekstu. 

                
var przywitajSię = function(imię) {
  console.log("Witaj, " + imię + ", jestem " + this.nazwa);
}.bind({nazwa: "ChatGPT"});

przywitajSię("Jan"); // Wyświetli: "Witaj, Jan, jestem Ignacy"
Uwaga

Te właściwości pozwalają na bardziej zaawansowane wykorzystywanie funkcji w języku JavaScript.
