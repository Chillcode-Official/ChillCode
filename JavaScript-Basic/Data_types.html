Typy Prymitywne

Typy prymitywne w języku JavaScript to typy danych, które są podstawowymi rodzajami danych w tym języku. Są one niezmiennymi wartościami, które nie są obiektami i nie posiadają żadnych metod ani właściwości.

String - ciąg znaków, np. "Hello World"

Number - liczby, np. 42 lub 3.14

										
var name = "John Doe";

console.log(name.length);
8 
										
var age = 30;

console.log(age + 5);
35

Boolean - wartość logiczna, true lub false

Undefined - wartość niezdefiniowana

										
var isValid = true;

console.log(!isValid);
false 
										
var x;

console.log(typeof x);
undefined

Null - wartość pusta 

								
var y = null;
console.log(typeof y); // (null jest traktowany jako obiekt w JavaScript)
object

Typy Odniesienia

Typy odniesienia (ang. reference types) w języku JavaScript to typy danych, które są reprezentowane jako obiekty i których wartości są przechowywane w pamięci jako odniesienia do miejsca, gdzie znajdują się ich wartości. Wartość typów odniesienia może być zmieniona, a same typy posiadają właściwości i metody.

Object - obiekty, np. {key1: value1, key2: value2}

Function - funkcje, np. function add(a, b) {return a + b;}

										
var person = {firstName: "John", lastName: "Doe"};

console.log(person.firstName);
John 
										
function add(a, b) {return a + b;}

console.log(add(2, 3));
5

Array - tablice, np. [1, 2, 3, 4, 5]

Date - obiekt daty i czasu, np. new Date()

										
var numbers = [1, 2, 3, 4, 5];
console.log(numbers[2]);
3 
										
var now = new Date();
console.log(now.getMonth()); // (aktualny miesiąc, 0-based)
2

RegExp - wyrażenia regularne, np. /pattern/ 

								
var pattern = /\d+/;
console.log(pattern.test("I have 30 apples."));
true
