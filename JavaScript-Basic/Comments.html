Komentarze

Komentarze w JavaScript są używane do opisywania intencji kodu i ułatwiania jego zrozumienia przez innych programistów lub przez samego siebie w przyszłości. Komentarze pozwalają także na wyjaśnienie działania poszczególnych fragmentów kodu, co jest szczególnie przydatne w przypadku skomplikowanych algorytmów.

Komentarze w JavaScript to między innymi komentarz liniowy oraz wieloliniowy

Komentarz liniowy 
				
// To jest komentarz liniowy
Komentarz wieloliniowy 
				
/*
To jest przykładowy
komentarz
wieloliniowy
*/
Note

Komentarze w języku JavaScript są bardzo ważne i powinny być dodawane tam, gdzie jest to konieczne, aby ułatwić zrozumienie i zwiększyć czytelność kodu.
