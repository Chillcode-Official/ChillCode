Funkcje Matematyczne

Funkcje matematyczne w języku JavaScript to funkcje dostępne w bibliotece Math, które umożliwiają wykonywanie różnych obliczeń matematycznych, takich jak obliczanie pierwiastka, potęgi, logarytmu, trigonometrii i wiele innych. Są one dostępne jako metody obiektu Math i można je wywoływać w swoim kodzie, aby wykonać określone obliczenia.

Przykład użycia kilku funkcji matematycznych w języku JavaScript: 

           
// Obliczenie wartości bezwzględnej liczby
console.log(Math.abs(-5)); // 5


// Zaokrąglenie liczby do najbliższej całkowitej
console.log(Math.round(4.7)); // 5
console.log(Math.round(4.4)); // 4


// Obliczenie największej i najmniejszej z kilku liczb
console.log(Math.max(1, 2, 3, 4, 5)); // 5
console.log(Math.min(1, 2, 3, 4, 5)); // 1


// Potęgowanie
console.log(Math.pow(2, 3)); // 8


// Obliczenie pierwiastka kwadratowego
console.log(Math.sqrt(16)); // 4


// Losowanie liczby zmiennoprzecinkowej z zakresu (0, 1)
console.log(Math.random());

Składnia użycia funkcji matematycznych jest następująca: 

        
Math.nazwa_funkcji(argumenty);
Wyjaśnienie

Gdzie nazwa_funkcji jest nazwą funkcji matematycznej, a argumenty to wartości, które są wprowadzane do funkcji jako argumenty i są używane do wykonywania obliczeń. Funkcje matematyczne zwracają wartość, która jest wynikiem obliczeń.

Najpopularniejsze Funkcję Matematyczne

Oto lista najpopularniejszych funkcji matematycznych dostępnych w języku JavaScript wraz z przykładami użycia:

Math.abs(x)- zwraca wartość bezwzględną liczby x.

Math.ceil(x)- zwraca najmniejszą liczbę całkowitą większą lub równą x.

          
Math.abs(-5); // 5
          
        
           
Math.ceil(4.2); // 5

Math.floor(x)- zwraca największą liczbę całkowitą mniejszą lub równą x.

Math.max(a,b)- zwraca największą wartość spośród argumentów a, b..

          
Math.floor(4.9); // 4
          
        
          
Math.max(1, 2, 3, 4); // 4

Math.min(a,b)- zwraca najmniejszą wartość spośród argumentów a, b..

Math.pow(x, y)- zwraca wartość x podniesione do potęgi y.

          
Math.min(1, 2, 3, 4); // 1
          
        
          
Math.pow(2, 3); // 8

Math.round(x)- zwraca najbliższą liczbę całkowitą dla liczby x.

Math.sqrt(x)- zwraca pierwiastek kwadratowy liczby x.

          
Math.round(4.5); // 5
         
        
          
Math.sqrt(16); // 4

Math.random()- zwraca losową liczbę z przedziału [0, 1).

Math.sin(x)- zwraca sinus kąta x wyrażonego w radianach.

          
Math.random(); // 0.123456789
         
        
          
Math.sin(Math.PI/2); // 1

Math.cos(x)- zwraca kosinus kąta x wyrażonego w radianach.

Math.log(x)- zwraca logarytm naturalny liczby x.

          
Math.cos(Math.PI); // -1
         
        
          
Math.log(10); // 2.302585092994046
Uwaga

Te przykłady pokazują, jak użyć funkcji matematycznych w języku JavaScript do wykonywania różnych obliczeń.
