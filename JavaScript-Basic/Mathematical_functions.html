Funkcje Matematyczne

Funkcje matematyczne w języku JavaScript to funkcje dostępne w bibliotece Math, które umożliwiają wykonywanie różnych obliczeń matematycznych, takich jak obliczanie pierwiastka, potęgi, logarytmu, trigonometrii i wiele innych. Są one dostępne jako metody obiektu Math i można je wywoływać w swoim kodzie, aby wykonać określone obliczenia.

Przykład użycia kilku funkcji matematycznych w języku JavaScript: 

 					
// Obliczenie wartości bezwzględnej liczby
console.log(Math.abs(-5)); // 5


// Zaokrąglenie liczby do najbliższej całkowitej
console.log(Math.round(4.7)); // 5
console.log(Math.round(4.4)); // 4


// Obliczenie największej i najmniejszej z kilku liczb
console.log(Math.max(1, 2, 3, 4, 5)); // 5
console.log(Math.min(1, 2, 3, 4, 5)); // 1


// Potęgowanie
console.log(Math.pow(2, 3)); // 8


// Obliczenie pierwiastka kwadratowego
console.log(Math.sqrt(16)); // 4


// Losowanie liczby zmiennoprzecinkowej z zakresu (0, 1)
console.log(Math.random());

Składnia użycia funkcji matematycznych jest następująca: 

				
Math.nazwa_funkcji(argumenty);
Wyjaśnienie

nazwa_funkcji jest nazwą funkcji matematycznej, a argumenty to wartości, które są wprowadzane do funkcji jako argumenty i są używane do wykonywania obliczeń. Funkcje matematyczne zwracają wartość, która jest wynikiem obliczeń.
