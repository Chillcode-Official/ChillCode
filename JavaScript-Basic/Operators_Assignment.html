Operatory Przypisania

Operatory przypisania w języku JavaScript pozwalają na przypisanie wartości do zmiennych. W języku JavaScript istnieją również inne operatory przypisania, takie jak <<=, >>=, >>>=, &=, ^= i |=, ale są one mniej często używane. O to kilka przykładów operatorów przypisania:

= Przypisuje wartość do zmiennej.

+= Dodaje wartość do zmiennej i zapisuje wynik z powrotem do zmiennej.

										
let x = 10;
x += 5;
console.log(x);
15 
										

let x = 10;

-= Odejmuje wartość od zmiennej i zapisuje wynik z powrotem do zmiennej.

*= Mnoży wartość zmiennej i zapisuje wynik z powrotem do zmiennej.

										
let x = 10;
x -= 5;
console.log(x);
5 
										
let x = 10;
x *= 5;
console.log(x);
50

/= Dzieli wartość zmiennej i zapisuje wynik z powrotem do zmiennej.

%= Oblicza resztę z dzielenia wartości zmiennej i zapisuje wynik z powrotem do zmiennej.

										
let x = 10;
x /= 5;
console.log(x);
2 
										
let x = 10;
x %= 3;
console.log(x);
1
