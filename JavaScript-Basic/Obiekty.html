(Obiekty)

Obiekty w języku JavaScript to jedna z podstawowych konstrukcji programowania. Obiekty służą do tworzenia i przechowywania zmiennych oraz funkcji, które są związane ze sobą w jednym miejscu. Obiekty w języku JavaScript są typem dynamicznym i nie wymagają określania ich typu przed użyciem.

Obiekty są tworzone za pomocą nawiasów klamrowych i składają się z właściwości i metod. Właściwości to zmienne przypisane do obiektu, a metody to funkcje, które są częścią obiektu. 

                
let person = {

  firstName: "John",
  lastName: "Doe",
  age: 30,

  getFullName: function() {
                    
    return this.firstName + " " + this.lastName;
  }
};
Wyjasnienie

W powyższym przykładzie person jest obiektem, który posiada cztery właściwości: firstName, lastName, age i metodę getFullName. Właściwości i metody są odwoływane za pomocą operatora ., np. person.firstName lub person.getFullName().

(Konstruktory) tworzenie obiektów

Konstruktory to funkcje w języku JavaScript, które służą do tworzenia nowych obiektów z określonymi właściwościami i metodami. Konstruktory są bardzo przydatne w tworzeniu wielu instancji obiektów o podobnych właściwościach i metodach.

W konstruktorze zazwyczaj używa się słowa kluczowego this, aby odwołać się do właściwości i metod, które należą do tworzonego obiektu. Właściwości i metody te są zazwyczaj przekazywane jako parametry konstruktora. 

                
function Person(name, age) {

  this.name = name;
  this.age = age;

  this.introduce = function() {

    console.log("My name is " + this.name + " and I am " + this.age + " years old.");
  };
}


var john = new Person("John", 30);
var jane = new Person("Jane", 25);


john.introduce(); // My name is John and I am 30 years old.
jane.introduce(); // My name is Jane and I am 25 years old.
Wyjasnienie

W powyższym przykładzie funkcja Person służy jako konstruktor do tworzenia obiektów john i jane, które mają właściwości name i age, oraz metodę introduce.

Aby utworzyć nowy obiekt za pomocą konstruktora, użyj słowa kluczowego new przed nazwą konstruktora. W powyższym przykładzie john i jane to nowe obiekty, które zostały utworzone za pomocą konstruktora Person.

Uwaga

Korzystanie z konstruktorów pozwala na tworzenie wielu instancji obiektów o podobnych właściwościach i metodach, co ułatwia tworzenie i zarządzanie aplikacją.

(Object.create) tworzenie obiektów

Metoda Object.create() to jedna z metod w języku JavaScript, która służy do tworzenia nowych obiektów z określonymi właściwościami i prototypami. W przeciwieństwie do konstruktora, metoda Object.create() tworzy nowy obiekt bez wywoływania żadnej funkcji konstruktora. Zamiast tego, jako argumenty metody przekazuje się obiekt prototypu i opcjonalny obiekt z właściwościami.

Oto przykład użycia metody Object.create() do utworzenia nowego obiektu: 

                
let personPrototype = {

  introduce: function() {
    console.log("My name is " + this.name + " and I am " + this.age + " years old.");
  }
};


let john = Object.create(personPrototype, {
  name: {
    value: "John"
  },

  age:{
    value: 30
  }
});

john.introduce(); // My name is John and I am 30 years old.
Wyjasnienie

W powyższym przykładzie utworzono prototyp personPrototype, który zawiera metodę introduce. Następnie, utworzono nowy obiekt john za pomocą metody Object.create(), w której jako pierwszy argument przekazano prototyp personPrototype, a jako drugi argument obiekt z właściwościami name i age.

Metoda Object.create() tworzy nowy obiekt john z właściwościami name i age, które zostały przekazane jako drugi argument. Obiekt john dziedziczy również metodę introduce z prototypu personPrototype.

Uwaga

Korzystanie z metody Object.create() pozwala na tworzenie nowych obiektów z określonymi prototypami i właściwościami bez konieczności tworzenia funkcji konstruktora. Jest to szczególnie przydatne, gdy potrzebujemy utworzyć wiele instancji obiektów o podobnych właściwościach i prototypach.
