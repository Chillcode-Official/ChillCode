Comments - Komentarze

Komentarze w języku JavaScript są używane do opisywania intencji kodu i ułatwiania jego zrozumienia przez innych programistów lub przez samego siebie w przyszłości. Komentarze pozwalają także na wyjaśnienie działania poszczególnych fragmentów kodu, co jest szczególnie przydatne w przypadku skomplikowanych algorytmów.

Komentarz jedno linijkowy jest oznaczany znakiem // i kończy się wraz z końcem linii: 

        
// To jest komentarz jedno linijkowy

Komentarz wielolinijkowy jest oznaczany znakami /* oraz */ 

        
/*
To jest komentarz
wielolinijkowy
*/
Uwaga

Komentarze w języku JavaScript są bardzo ważne i powinny być dodawane tam, gdzie jest to konieczne, aby ułatwić zrozumienie i zwiększyć czytelność kodu.
