Eventy

Eventy w języku JavaScript to zdarzenia wywoływane przez użytkownika lub przez przeglądarkę w odpowiedzi na określone akcje. Zdarzenia te mogą być związane z interakcją użytkownika z aplikacją, takimi jak kliknięcia, wprowadzanie danych, przesuwanie myszką lub zmiana rozmiaru okna przeglądarki. Można je obsłużyć za pomocą kilku sposobów, takich jak:

Wykorzystanie wbudowanej w przeglądarkę funkcji addEventListener 

                
document.getElementById("myBtn").addEventListener("click", function(){
  alert("Button was clicked");
});

Użycie skróconej wersji addEventListener 

                
document.getElementById("myBtn").onclick = function(){
  alert("Button was clicked");
};

Użycie atrybutu on-event 

                
<button id="myBtn" onclick="alert('Button was clicked')">Click me</button>

Te trzy sposoby są do siebie podobne, jednak zaleca się stosowanie pierwszej metody, ponieważ pozwala ona na dodawanie wielu obsług zdarzeń do jednego elementu, co jest trudne do osiągnięcia przy użyciu dwóch pozostałych sposobów.

Obsługa Zdarzeń

Obsługa zdarzeń jest bardzo prosta i polega na dodaniu do elementu HTML funkcji, która będzie wykonywana w momencie zaistnienia określonego zdarzenia. W celu dodania funkcji obsługującej zdarzenie można skorzystać z wyżej wspomnianych trzech sposobów.

Niektóre ze zdarzeń, które można obsłużyć w języku JavaScript.

  • click Kliknięcie przycisku
  • change Zmiana wartości pola tekstowego
  • keypress Naciśnięcie klawisza na klawiaturze
  • mousemove Przesuwanie kursora myszki
  • load Załadowanie strony
  • resize Zmiana rozmiaru okna przeglądarki
Uwaga

Pamiętaj, że obsługa zdarzeń jest jednym z najważniejszych aspektów tworzenia interaktywnych stron internetowych i jest bardzo pomocna w codziennym życiu programisty. Dlatego warto poświęcić czas na zgłębienie tego tematu.

Właściwości Eventów

W języku JavaScript każde zdarzenie jest reprezentowane przez obiekt Event i posiada szereg właściwości opisujących jego stan i szczegóły. Poniżej wymieniono niektóre z nich:

type Zwraca nazwę zdarzenia. 

                
document.addEventListener("click", function(event) {
  console.log(event.type);
});

target Zwraca element, który wywołał zdarzenie. 

                
document.addEventListener("click", function(event) {
  console.log(event.target);
});

clientX oraz clientY Zwracają pozycję kursora myszy w oknie przeglądarki. 

                
document.addEventListener("mousemove", function(event) {
  console.log(`X: ${event.clientX}, Y: ${event.clientY}`);
});

keyCode Zwraca kod klawisza, który został naciśnięty. 

                
document.addEventListener("keydown", function(event) {
  console.log(event.keyCode);
});

preventDefault() Zapobiega domyślnej akcji przeglądarki dla danego zdarzenia. 

                
document.querySelector("a").addEventListener("click", function(event) {
  event.preventDefault();
  console.log("Link was clicked, default action was prevented");
});

stopPropagation() Zatrzymuje propagowanie zdarzenia do innych elementów w hierarchii dokumentu. 

                
document.addEventListener("click", function(event) {
  console.log("Document was clicked");
  event.stopPropagation();
});

document.querySelector("button").addEventListener("click", function(event) {
  console.log("Button was clicked");
});
Uwaga

Oto tylko kilka przykładów, jak można używać właściwości obiektu Event w języku JavaScript. W zależności od potrzeb i typu zdarzenia mogą być dostępne różne właściwości i metody.
