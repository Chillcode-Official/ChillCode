Pierwszy program

Program "Witaj świecie!" jest podstawowym programem, od którego zaczyna każdy początkujący programista. Oto przykładowy program w języku JavaScript, który wyświetla na ekranie napis "Witaj świecie!" i kończy swoje działanie:

Program, który wypisze tekst w konsoli:

                    
console.log("Witaj świecie!");
Witaj świecie!

Aby uruchomić ten program, można skorzystać z konsoli przeglądarki internetowej lub środowiska programistycznego (np. Visual Studio Code). W konsoli należy wpisać powyższy kod i nacisnąć klawisz Enter. Program powinien wyświetlić tekst "Witaj świecie!" w konsoli.

Wyjaśnienie

Kod console.log("Witaj świecie!"); składa się z dwóch elementów: console.log oraz "Witaj świecie!". console.log jest metodą obiektu console, który jest częścią przeglądarki internetowej lub środowiska programistycznego, w którym uruchamiasz kod. Metoda ta służy do wyświetlania informacji w konsoli, czyli specjalnym oknie, w którym programiści mogą śledzić informacje o działaniu swojego kodu. "Witaj świecie!" jest ciągiem znaków, zwany również łańcuchem znaków lub napisem, który zostanie wyświetlony w konsoli.

Note

Zatem kod console.log("Witaj świecie!"); oznacza, że ​​chcesz wyświetlić napis "Witaj świecie!" w konsoli. Funkcja console.log służy do wyświetlania informacji w konsoli, a łańcuch znaków "Witaj świecie!" to właśnie informacja, którą chcesz wyświetlić.
