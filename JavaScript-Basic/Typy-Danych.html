Typy Prymitywne JavaScript

Typy prymitywne (primitive-types) w języku JavaScript to typy danych, które są podstawowymi rodzajami danych w tym języku. Są one niezmiennymi wartościami, które nie są obiektami i nie posiadają żadnych metod ani właściwości.

String - ciąg znaków, np. "Hello World"

Number - liczby, np. 42 lub 3.14

                    
var name = "John Doe";

console.log(name.length); // Output: 8

                    
                
                    
var age = 30;

console.log(age + 5); // Output: 35

Boolean - wartość logiczna, true lub false

Undefined - wartość niezdefiniowana

                    
var isValid = true;

console.log(!isValid); // Output: false
                    
                
                    
var x;

console.log(typeof x); // Output: undefined

Null - wartość null 

                
var y = null;
console.log(typeof y); // Output: object (null jest traktowany jako obiekt w JavaScript)

Typy Odniesienia JavaScript

Typy odniesienia (ang. reference types) w języku JavaScript to typy danych, które są reprezentowane jako obiekty i których wartości są przechowywane w pamięci jako odniesienia do miejsca, gdzie znajdują się ich wartości. Wartość typów odniesienia może być zmieniona, a same typy posiadają właściwości i metody.

Object - obiekty, np. {key1: value1, key2: value2}

Function - funkcje, np. function add(a, b) {return a + b;}

                    
var person = {firstName: "John", lastName: "Doe"};

console.log(person.firstName); // Output: John
                    
                
                    
function add(a, b) {return a + b;}

console.log(add(2, 3)); // Output: 5

Array - tablice, np. [1, 2, 3, 4, 5]

Date - obiekt daty i czasu, np. new Date()

                    
var numbers = [1, 2, 3, 4, 5];
console.log(numbers[2]); // Output: 3
                    
                
                    
var now = new Date();
console.log(now.getMonth()); // Output: (aktualny miesiąc, 0-based)

RegExp - wyrażenia regularne, np. /pattern/ 

                
var pattern = /\d+/;
console.log(pattern.test("I have 30 apples.")); // Output: true
