Zmienne JavaScript

Zmienna w języku JavaScript to nazwane miejsce w pamięci komputera, które służy do przechowywania danych. Zmienne są używane do przechowywania informacji, takich jak liczby, teksty, obiekty i inne typy danych.

Aby utworzyć zmienną w języku JavaScript, należy użyć słowa kluczowego var, let lub const sekwencji (równa się) i wartości, którą chcemy przypisać do zmiennej.

Zmienna Let

let jest słowem kluczowym w języku JavaScript, które służy do tworzenia zmiennych. let jest używane do przypisywania wartości do zmiennej, która ma zasięg blokowy. To znaczy, że zmienna utworzona przy użyciu let jest widoczna tylko wewnątrz bloku, w którym została utworzona.

Poniższy kod tworzy zmienną o nazwie name i przypisuje jej wartość "John Doe" przy użyciu słowa kluczowego let. Następnie, w następnej linijce, wartość tej zmiennej jest nadpisywana i ustawiana na "Kai Doe". W ostatniej linijce kodu wywoływana jest funkcja console.log i wyświetlana jest wartość zmiennej name, co w tym przypadku jest "Kai Doe". 

                
let name = "John Doe";

name = "Kai Doe";

console.log(name);

//output: Kai Doe

Zmienna Const

const jest słowem kluczowym w języku JavaScript, które służy do tworzenia stałych zmiennych. Stała zmienna jest zmienną, której wartość nie może być zmieniona po jej utworzeniu.

Używamy słowa kluczowego const aby utworzyć zmieną "name" 

                
const Pi = 3.14;

Próba zmiany wartości stałej zmiennej powoduje błąd. 

                
Pi = 5;

//output: Uncaught TypeError: Assignment to constant variable.

Zmienna Var

var jest słowem kluczowym w języku JavaScript, które służy do tworzenia zmiennych. var jest używane do przypisywania wartości do zmiennej i ma zasięg funkcyjny. To znaczy, że zmienna utworzona przy użyciu var jest widoczna wewnątrz funkcji, w której została utworzona, oraz poza nią.

Poniższy kod tworzy zmienną o nazwie name i przypisuje jej wartość "John Doe". Następnie, w następnej linijce, wartość tej zmiennej jest nadpisywana i ustawiana na "Kai Doe". W ostatniej linijce kodu wywoływana jest funkcja console.log i wyświetlana jest wartość zmiennej name, co w tym przypadku jest "Kai Doe". 

                
var name = "John Doe";

name = "Kai Doe";

console.log(name);

//output: Kai Doe
Uwaga

Z uwagi na swój zasięg funkcyjny, var często jest uważane za przestarzałe i zastąpione przez let i const, które oferują bardziej zaawansowane i precyzyjne zasięgi zmiennych.
