Argumenty

W języku C++, argumenty to wartości przekazywane do funkcji podczas jej wywołania. Funkcja może przyjmować jeden lub więcej argumentów, które określają informacje, na podstawie których funkcja będzie działać.

Przykładowo, to jest funkcja dodawania dwóch liczb: 

								
int dodaj(int a, int b) {
		return a + b;
}

Funkcja ta przyjmuje dwa argumenty typu int o nazwach a i b. Gdy funkcja jest wywoływana, wartości przekazywane jako argumenty zostaną przypisane do zmiennych a i b. Na przykład, jeśli wywołujemy tę funkcję w ten sposób: 

								
int wynik = dodaj(3, 5);

Zmienna wynik będzie miała wartość 8, ponieważ funkcja dodaj zostanie wywołana z argumentami 3 i 5, a następnie zwróci ich sumę.

Notatka

Argumenty mogą mieć różne typy, w zależności od potrzeb funkcji. Funkcja może również przyjmować argumenty opcjonalne, które nie muszą być przekazywane podczas wywoływania funkcji.
