Zmienne

W C++, zmienne są deklarowane, aby przechowywać dane. Każda zmienna ma swój typ, który określa, jakiego rodzaju dane będą przechowywane, np. liczby całkowite, liczby zmiennoprzecinkowe, znaki, itp.

Oto kilka przykładów deklaracji zmiennych w C++: 

                
int liczba_całkowita;   // zmienna typu int przechowująca liczby całkowite

float liczba_zmiennoprzecinkowa;   // zmienna typu float przechowująca liczby zmiennoprzecinkowe

char znak;   // zmienna typu char przechowująca pojedynczy znak

bool logiczna;   // zmienna typu bool przechowująca wartość logiczną (true/false)

W C++ można również zainicjować zmienną podczas jej deklaracji 

                 
int liczba_całkowita = 10;   // zmienna typu int zainicjowana wartością 10

float liczba_zmiennoprzecinkowa = 3.14;   // zmienna typu float zainicjowana wartością 3.14

char znak = 'a';   // zmienna typu char zainicjowana wartością 'a'

bool logiczna = true;   // zmienna typu bool zainicjowana wartością true

W C++ można również wykorzystać zmienne do przetwarzania danych 

                
int a = 10;

int b = 5;

int suma = a + b;   // zmienna "suma" przechowuje sumę wartości zmiennych "a" i "b"

cout << "Suma: " << suma << endl;   // wyświetla wynik na ekranie


//output: 15
Uwaga

Zmienne w C++ są deklarowane, aby przechowywać dane. Każda zmienna ma swój typ i można ją zainicjować wartością podczas deklaracji. Zmienne są również wykorzystywane do przetwarzania danych w programach.
