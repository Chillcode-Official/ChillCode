Pętla for

Pętla for w C++ jest jednym z najczęściej używanych typów pętli i pozwala na powtarzanie bloków kodu przez określoną liczbę razy.

Składnia pętli for wygląda następująco: 

                
for (inicjalizacja; warunek; zwiększenie)
{
  // blok kodu, który zostanie wykonany w każdej iteracji
}

Składnia pętli:

  • inicjalizacja- sekcja ta inicjuje zmienne używane w pętli, zazwyczaj deklaruje się tu licznik pętli, czyli zmienną, która będzie używana do śledzenia ilości iteracji.

  • warunek- warunek, który musi być spełniony, aby pętla kontynuowała działanie. Gdy warunek jest fałszywy, pętla kończy swoje działanie.

  • zwiększenie- sekcja ta definiuje, jak wartość zmiennej zliczającej pętle jest zwiększana w każdej iteracji.

Przykład użycia pętli for, Ten kod wyświetli liczby od 0 do 9 w konsoli. 

                
for (int i = 0; i < 10; i++)
{
  cout << i << std::endl;
}

Można również użyć pętli for do iteracji po tablicach, wektorach lub innych kolekcjach danych. W takim przypadku można użyć iteratora w sekcji warunku i zwiększenia. 

                
vector<int> vec = { 1, 2, 3, 4, 5 };

for (auto it = vec.begin(); it != vec.end(); ++it)
{
  cout << *it << endl;
}

Pętla While

Pętla while w języku C++ to konstrukcja pozwalająca na powtarzanie bloku kodu, dopóki określony warunek jest spełniony. Ogólna składnia pętli while wygląda następująco:

Składnia pętli while wygląda następująco: 

                
while (warunek)
{
  // blok kodu, który zostanie wykonany dopóki warunek jest spełniony
}

Składnia pętli:

  • warunek- warunek, który musi być spełniony, aby pętla kontynuowała działanie. Gdy warunek jest fałszywy, pętla kończy swoje działanie.

Przykład pętli do while iterującej przez liczby od 1 do 5, Wynik działania powyższego kodu to wyświetlenie na ekranie liczb od 1 do 5. 

                
int i = 1;

while (i <= 5) {
  std::cout << i << std::endl;
  i++;
}

Pętla while jest szczególnie przydatna w przypadku, gdy liczba iteracji nie jest znana z góry, a zależy od jakiegoś warunku lub zmiennej. Poniżej znajduje się przykład pętli while wykorzystującej losową liczbę iteracji: 

                
#include <iostream>
#include <cstdlib>
#include <ctime>

int main() {
  std::srand(std::time(nullptr)); // inicjalizacja generatora liczb pseudolosowych
  int i = 0;
  int max_iter = std::rand() % 10 + 1; // losowa liczba iteracji od 1 do 10

  while (i < max_iter) {
    std::cout << "Iteracja nr " << i+1 << std::endl;
    i++;
  }

  return 0;
}
Uwaga

W tym przykładzie pętla do while będzie wykonywana przynajmniej raz, a następnie będzie kontynuowana dopóki użytkownik nie zgadnie liczby losowej.

Pętla Do..While

Pętla do while w języku C++ to podobna do pętli while konstrukcja, która również pozwala na powtarzanie bloku kodu dopóki określony warunek jest spełniony. Jednak w odróżnieniu od pętli while, blok kodu jest wykonywany przynajmniej raz, niezależnie od tego, czy warunek jest spełniony czy nie.

Składnia pętli do..while wygląda następująco: 

                
do {
  // blok kodu, który zostanie wykonany przynajmniej raz
} while (warunek);

Składnia pętli:

  • warunek- warunek, który musi być spełniony, aby pętla kontynuowała działanie. Gdy warunek jest fałszywy, pętla kończy swoje działanie.

Przykład pętli do while iterującej przez liczby od 1 do 5, Wynik działania powyższego kodu to wyświetlenie na ekranie liczb od 1 do 5.

Pętla do while jest przydatna w przypadku, gdy chcemy wykonać pewien blok kodu przynajmniej raz, a następnie kontynuować dopóki określony warunek jest spełniony. Przykład poniżej ilustruje to na przykładzie prostej gry, w której użytkownik musi zgadnąć losową liczbę z przedziału od 1 do 10: 

                
#include <iostream>
#include <cstdlib>
#include <ctime>

int main() {
  std::srand(std::time(nullptr)); // inicjalizacja generatora liczb pseudolosowych
  int liczba_losowa = std::rand() % 10 + 1;
  int liczba;

  do {
    std::cout << "Podaj liczbę od 1 do 10: ";
    std::cin >> liczba;

    if (liczba < liczba_losowa) {
      std::cout << "Za mało!" << std::endl;
    } else if (liczba > liczba_losowa) {
      std::cout << "Za dużo!" << std::endl;
    } else {
      std::cout << "Brawo! Zgadłeś!" << std::endl;
    }
  } while (liczba != liczba_losowa);

  return 0;
}
Uwaga

W tym przykładzie pętla do while będzie wykonywana przynajmniej raz, a następnie będzie kontynuowana dopóki użytkownik nie zgadnie liczby losowej.

Pętla forEach

Pętla forEach, zwana również pętlą for-each lub pętlą zakresową, to konstrukcja, która pozwala na iterowanie przez elementy kolekcji, np. tablicy, listy czy wektora w języku C++. Pętla ta jest dostępna od wersji języka C++11 i opiera się na użyciu tzw. iteracji po zakresie (range-based loop).

Składnia pętli forEach wygląda następująco: 

                
for (typ_zmiennej zmienna : kolekcja) {
  // blok kodu, który zostanie wykonany dla każdego elementu kolekcji
}
  • typ zmiennej- typ zmiennej, która będzie przechowywać kolejne elementy kolekcji,
  • zmienna- nazwa zmiennej, która będzie przechowywać kolejne elementy kolekcji,
  • kolekcja- kolekcja, po której iterujemy.

Przykład pętli forEach iterującej przez elementy tablicy, W powyższym przykładzie pętla forEach iteruje przez elementy tablicy tablica i wyświetla każdy z nich na ekranie. 

                
int tablica[] = {1, 2, 3, 4, 5};

for (int liczba : tablica) {
  std::cout << liczba << std::endl;
}

Pętla forEach jest przydatna, gdy chcemy szybko i łatwo iterować przez elementy kolekcji i wykonać na nich pewne operacje. Przykład poniżej ilustruje to na przykładzie sumowania elementów wektora, W tym przykładzie pętla forEach iteruje przez elementy wektora wektor i sumuje ich wartości. 

                
#include <iostream>
#include <vector>

int main() {
  std::vector<int> wektor = {1, 2, 3, 4, 5};
  int suma = 0;

  for (int liczba : wektor) {
    suma += liczba;
  }

  std::cout << "Suma elementów wektora: " << suma << std::endl;

  return 0;
}
