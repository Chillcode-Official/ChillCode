Pierwszy Program C++

Program "Hello, World" jest podstawowym programem od którego zaczyna każdy początkujący programista. Oto przykładowy program w języku C++, który wyświetla na ekranie napis "Hello, World!" i kończy swoje działanie: 

        
#include <iostream>

int main() {

    std::cout << "Hello, World!\n";

    return 0;
}

Wyjaśnienie

Po uruchomieniu tego programu na konsoli pojawi się napis "Witaj, świecie!".

Obiekt cout jest zdefiniowany w nagłówku iostream i służy do wyprowadzania danych na konsolę.

Funkcja main() jest punktem wejścia do programu

instrukcja return 0; służy do wskazania, że program zakończył się pomyślnie.
