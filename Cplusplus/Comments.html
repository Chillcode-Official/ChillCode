Komentarze

W języku C++, komentarze służą do opisywania kodu w sposób zrozumiały dla ludzi, komentarze są ignorowane przez kompilator, co oznacza, że nie są wliczane w działanie programu. Ich jedyną funkcją jest dostarczenie informacji dla programisty lub innych osób, które czytają kod.

Komentarze w c++ to między innymi komentarz Jednolinijkowy oraz Wielolinijkowy

Jedno linijkowy

        
// To jest jednolinijkowy komentarz

int liczba = 5; // To jest też jednolinijkowy komentarz

Wielolinijkowe

        
/*
To jest wielolinijkowy komentarz.
Można go użyć do opisu większej ilości kodu.
*/

int liczba = 5; /* Można też użyć jednolinijkowego komentarza w środku kodu */
Ważne

Komentarze są bardzo ważne, ponieważ pozwalają na czytelniejsze zrozumienie kodu, szczególnie dla innych programistów, którzy mogą pracować nad tym samym kodem w przyszłości.
