Typy Danych

W C++ typy danych określają rodzaj danych, które mogą być przechowywane w zmiennych. Typ danych określa zakres wartości, jakie mogą być przechowywane w zmiennej, jakie operacje można wykonywać na zmiennej i jak wiele pamięci jest potrzebne do przechowywania wartości zmiennej.

Przykłady użycia różnych typów danych w C++:

Przechowują wartości całkowite, takie jak -3, 0, 100 itp. W C++ są dostępne różne typy całkowite, w tym: int, short, long, long long. 

                                
int liczba = 10;
int wynik = liczba + 5;
cout << "Wynik: " << wynik << endl;

Przechowują liczby zmiennoprzecinkowe, takie jak 3,14 lub 2,71828. W C++ dostępne są dwa typy zmiennoprzecinkowe: float (32 bity) i double (64 bity). 

                                
float n = 3.14;
double m = 2.71828;
double wynik = n * m;
cout << "Wynik: " << wynik << endl;

Przechowuje pojedynczy znak, takie jak 'a' lub '7'. Typ char przechowuje wartości całkowite z zakresu -128 do 127 lub 0 do 255, w zależności od tego, czy jest to typ char ze znakiem, czy bez znaku. 

                                
char litera = 'a';

cout << "Litera: " << litera << endl;

Przechowuje wartości logiczne true (prawda) lub false (fałsz). Typ bool jest często używany w instrukcjach warunkowych i pętlach. 

                                
bool prawda = true;

if (prawda) {
    cout << "Prawda!" << endl;
}
else {
    cout << "Fałsz!" << endl;
}

Przechowuje wartości zdefiniowane przez programistę. Typ wyliczeniowy jest przydatny w przypadku, gdy chcemy użyć nazwy zamiast liczby. 

                                
enum Kierunki {GORA, DOL, LEWO, PRAWO};
Kierunki kierunek = GORA;

if (kierunek == GORA){
    cout << "Kierunek: Góra" << endl;
}
                                
                            
                
#include <iostream>

int main() {

    short n = 5;      //liczba całkowita o małej pojemności

    int n = 10;       //liczba całkowita o średniej pojemności

    long n = 100;     //liczba całkowita o wysokiej pojemności

    long long n = 100;     //liczba całkowita o największej pojemności

    float = 2,13;      //liczba zmiennoprzecinkowa z pojedynczą precyzją,

    double = 2,71828;    //liczba zmiennoprzecinkowa z podwójną precyzją.

    char znak = 'Z';   //służy do przechowywania pojedynczych znaków

    bool prawda = true;   //typ logiczny prawda lub fałsz

}
