Instrukcje Warunkowe C#

Instrukcja warunkowa w języku C# pozwala na wykonywanie określonych akcji w zależności od wyniku sprawdzenia pewnych warunków od których będzie zależał dalszy przebieg programu.

W C# instrukcje warunkowe są zapisywane za pomocą słów kluczowych if oraz else

Instrukcja warunkowa if

Instrukcja warunkowa if jest używana do sprawdzenia warunku i wykonania określonych akcji w zależności od tego, czy warunek jest spełniony, czy nie. Składnia instrukcji if wygląda następująco:

Jeżeli wyrażenie logiczne jest prawdziwe zostanie wykonany kod wewnątrz bloku instrukcji. W przeciwnim razie program nie wykona kodu wewnątrz instrukcji a przejdzie do kolejnej liniki za klamrą zamykającą instrukcję warunkową. 

                
if (warunek)
{
    // kod do wykonania, jeśli warunek jest spełniony
}
                
            
                
using System;

class Program
{
    int x = 12;

    if (x > 8)
    {
      Console.WriteLine("x jest większe od 8");
    }
}

//output: x jest większe od 8

Instrukcja Warunkowa else...if else

Instrukcja else jest używana w połączeniu z instrukcją warunkową if do wykonania określonych akcji, jeśli warunek sprawdzony przy użyciu instrukcji if nie jest spełniony, program wykonuje instrukcje else.

Instrukcja else jest opcjonalna i może być pominięta, jeśli nie chcesz wykonywać żadnych akcji, jeśli warunek nie jest spełniony. 

                
using System;

class Program
{
    int wiek = 16;

    if (x >= 18)  //instrukcja if sprawdza czy zmienna wiek jest większa lub równa 18
    {
        Console.WriteLine("Jesteś pełnoletni (;");
    }

    else  //w tym przypadku nasza zmienna wiek była ponieżej 18 dlatego program wywoła instrukcje else
    {
        Console.WriteLine("Nie jesteś pełnoletni |:")
    }
}

//output: Nie jesteś pełnoletni |:

Instrukcja warunkowa else if jest używana w połączeniu z instrukcją if do sprawdzenia dodatkowych warunków. 

                
if (warunek1)
{
    // kod do wykonania, jeśli warunek1 jest spełniony
}
else if (warunek2)
{
    // kod do wykonania, jeśli warunek1 nie jest spełniony, ale warunek2 jest spełniony
}
else
{
    // kod do wykonania, jeśli żaden z warunków nie jest spełniony
}
Uwaga

Można użyć wielu instrukcji else if w jednym bloku instrukcji if, aby sprawdzić wiele warunków 

                
if (warunek1)
{
    // kod do wykonania, jeśli warunek1 jest spełniony
}
else if (warunek2)
{
    // kod do wykonania, jeśli warunek1 nie jest spełniony, ale warunek2 jest spełniony
}
else if (warunek3)
{
    // kod do wykonania, jeśli warunek1 i warunek2 nie są spełnione, ale warunek3 jest spełniony
}
else
{
    // kod do wykonania, jeśli żaden z warunków nie jest spełniony
}
Uwaga

Jeśli wyrażenie else if zostanie spełnione program nie wejdzie do żadnego innego warunku else if lub else 

                
using System;

class Program
{
    int x = 8;

    if (x > 8)
    {
        Console.WriteLine("x jest większe od 8");
    }
    else if (x == 8)
    {
        Console.WriteLine("x jest równe 8");
    }
    else
    {
        Console.WriteLine("x jest mniejsze od 8");
    }
}

//output: x jest równe 8

Zagnieżdzona Instrukcja if

nstrukcja warunkowa if może być zagnieżdżona w innej instrukcji if, co pozwala na sprawdzenie wielu warunków w ramach jednej instrukcji.

Można zagnieżdżać instrukcje if na dowolnie głębokim poziomie, ale należy zachować ostrożność, aby uniknąć zbyt skomplikowanego kodu, który może być trudny do zrozumienia i utrzymania. 

                
if (warunek1)
{
    // kod do wykonania, jeśli warunek1 jest spełniony

    if (warunek2)
    {
        // kod do wykonania, jeśli warunek2 jest spełniony
    }
}
else
{
    // kod do wykonania, jeśli warunek1 nie jest spełniony
}

Można również użyć instrukcji else if wewnątrz zagnieżdżonej instrukcji if 

                
if (warunek1)
{
    // kod do wykonania, jeśli warunek1 jest spełniony

    if (warunek2)
    {
        // kod do wykonania, jeśli warunek2 jest spełniony
    }
    else if (warunek3)
    {
        // kod do wykonania, jeśli warunek2 nie jest spełniony, ale warunek3 jest spełniony
    }
}
else
{
    // kod do wykonania, jeśli warunek1 nie jest spełniony
}

Przykład prostego programu używającego zagnieżdzenia instrukcji if 

                
using System;

class Program
{
    int x = 5;

    int y = 10;

    if (x > 0)
    {
        if (y > 0)
        {
            Console.WriteLine("x i y są dodatnie");
        }
        else
        {
            Console.WriteLine("x jest dodatnie, ale y jest ujemne");
        }
    }
    else
    {
        Console.WriteLine("x jest ujemne");
    }
}

//output: x i y są dodatnie

Instrukcja Wyboru Switch

Instrukcja wyboru switch jest używana do sprawdzania różnych wartości jednej zmiennej i wykonania określonych akcji w zależności od tej wartości.

Każda opcja w instrukcji switch jest sprawdzana od góry do dołu i kod jest wykonywany tylko dla pierwszej pasującej opcji. 

                
switch (zmienna)
{
    case wartosc1:
        // kod do wykonania, jeśli zmienna jest równa wartosc1
        break;
    case wartosc2:
        // kod do wykonania, jeśli zmienna jest równa wartosc2
        break;
    // itd.
    default:
        // kod do wykonania, jeśli żadna z powyższych wartości nie jest spełniona
        break;
}
Uwaga

Słowo kluczowe break jest używane do przerwania dalszego sprawdzania i przejścia do kodu po instrukcji switch.

Uwaga

Opcja default jest opcjonalna i jest wykonywana, jeśli żadna inna opcja nie zostanie spełniona.

  • Wyrażenie użyte w instrukcji wyboru musi być typu integralnego typu wyliczeniowego lub może być klasą, która pozwala na konwersję do typu integralnego lub wyliczeniowego.

  • Nie każdy przypadek musi zawierać instrukcję break.

  • Można zdefiniować dowolną liczbę przypadków.

Instrukcja switch jest szczególnie przydatna, gdy chcesz sprawdzić wiele różnych wartości jednej zmiennej i wykonać odpowiedni kod dla każdej z nich. 

                
using System;

class Program
{
    static void Main(string[] args)
     {
         int ocena = 5;

         switch (ocena)
         {
             case 6:
                 Console.WriteLine("Ocena celująca");
                 break;

             case 5:
                 Console.WriteLine("Ocena bardzo dobra");
                 break;

             case 4:
                 Console.WriteLine("Ocena dobra");
                 break;

             case 3:
                 Console.WriteLine("Ocena dostateczna");
                 break;

             case 2:
                 Console.WriteLine("Ocena Niedopuszczalna");
                 break;

             case 1:
                 Console.WriteLine("Ocena niedostateczna");
                 break;

             default:
                   Console.WriteLine("Nieznana ocena");
                   break;
         }
     }
}

//output: Ocena bardzo dobra

Zagnieżdzona Instrukcja Wyboru switch

instrukcja wyboru switch może być również zagnieżdżona w innej instrukcji switch lub instrukcji warunkowej if.

Zagnieżdżona instrukcja switch jest sprawdzana w taki sam sposób jak zwykła instrukcja switch, z tym że jest ona wykonywana tylko wtedy, gdy odpowiedni warunek został spełniony w instrukcji nadrzędnej. 

                
switch (zmienna1)
{
    case wartosc1:
        // kod do wykonania, jeśli zmienna1 jest równa wartosc1
        switch (zmienna2)
        {
            case wartosc3:
                // kod do wykonania, jeśli zmienna2 jest równa wartosc3
                break;

            case wartosc4:
                // kod do wykonania, jeśli zmienna2 jest równa wartosc4
                break;
        }
        break;

    case wartosc2:
        // kod do wykonania, jeśli zmienna1 jest równa wartosc2
        break;

    default:
        // kod do wykonania, jeśli żadna z powyższych wartości nie jest spełniona
        break;
}

Prosty przykład zagnieżdżonej instrukcji wyboru switch w języku C#. 

                
using System;

class Program
{
    static void Main(string[] args)
     {
         int ocena = 5;

         int wynik = 77;

         switch (ocena)
         {
             case 6:
                 Console.WriteLine("Ocena celująca");
                 break;

             case 5:
                 if (wynik >= 80)
                 {
                     Console.WriteLine("Ocena bardzo dobra");
                 }
                 else
                 {
                     Console.WriteLine("Ocena dobra");
                 }
                 break;

             case 4:
                 Console.WriteLine("Ocena dobra");
                 break;

             case 3:
                 Console.WriteLine("Ocena dostateczna");
                 break;

             case 2:
                 Console.WriteLine("Ocena Niedopuszczalna");
                 break;

             case 1:
                 Console.WriteLine("Ocena niedostateczna");
                 break;

             default:
                 Console.WriteLine("Nieznana ocena");
                 break;
         }
    }
}

//output: Ocena dobra
