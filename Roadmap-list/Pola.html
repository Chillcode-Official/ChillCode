Pola w Języku C#

Pola w języku C# to elementy składowe klasy lub struktury, które przechowują dane. Są one podobne do zmiennych, ale są one częścią klasy lub struktury i są dostępne dla wszystkich metod w klasie lub strukturze.

Oto przykład deklaracji pola w języku C# przy użyciu modyfikatora public

                
using System;

public class MojaKlasa

{
    public int _mojePole;
}

//output: 0

W tym przykładzie deklarowane jest pole _mojePole typu int w klasie MojaKlasa. Pole to jest publiczne, co oznacza, że jest dostępne dla wszystkich metod w klasie MojaKlasa oraz dla innych klas.

Prywatne Pola

Pola mogą być również deklarowane jako prywatne, co oznacza, że są dostępne tylko dla metod w klasie lub strukturze, w której zostały zdefiniowane.

Oto przykład deklaracji pola w języku C# przy użyciu modyfikatora private

                
using System;

public class MojaKlasa

{
    private int _mojePrywatnePole;
}

//output: 0

W tym przykładzie deklarowane jest pole _mojePrywatnePole typu int w klasie MojaKlasa. Pole to jest prywatne, co oznacza, że jest dostępne tylko dla metod w klasie MojaKlasa, inne klasy nie mają dostępu do tego pola.

Chronione Pola

Pola mogą być również deklarowane jako chronione co oznacza, że są dostępne tylko dla metod w klasie lub strukturze, w której zostały zdefiniowane, oraz dla klas pochodnych.

Oto przykład deklaracji pola w języku C# przy użyciu modyfikatora protected

                
using System;

public class MojaKlasa

{
    protected int _mojeChronionePole;
}

//output: 0

W tym przykładzie deklarowane jest pole _mojeChronionePole typu int w klasie MojaKlasa. Pole to jest chronione, co oznacza, że jest dostępne tylko dla metod w klasie MojaKlasa.
