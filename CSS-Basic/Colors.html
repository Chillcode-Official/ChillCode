Kolory W Języku CSS

CSS umożliwia definiowanie kolorów dla różnych elementów na stronie, np. tekstu, tła, obramowania itp. Kolory w CSS są definiowane za pomocą właściwości "color" lub "background-color". Kolory w CSS mogą być określane na kilka sposobów, w tym:

Kolory nazwane - nazwa koloru

Kolory RGB - czerwony, zielony, niebieski

										
p {

  color: blue;

}
										
								
										
p {

  color: rgb(0, 0, 255);

}

Kolory HEX - kod szesnastkowy

Kolory HSL - barwa, nasycenie, jasność

										
p {

  color: #0000FF;

}
										
								
										
p {

  color: hsl(240, 100%, 50%);

}

Przykład definiowania kolorów

								
p {
  color: blue;

  background-color: yellow;
}


h1 {
  color: rgb(255, 0, 0);

  background-color: rgb(0, 255, 0);
}


div {
  color: #FF0000;

  background-color: #00FF00;
}


span {
  color: hsl(0, 100%, 50%);

  background-color: hsl(120, 100%, 50%);
}
