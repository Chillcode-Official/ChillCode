Tło W Języku CSS

CSS (Cascading Style Sheets) pozwala na definiowanie wyglądu strony internetowej, w tym także tła. Można ustawić tło dla całej strony lub dla poszczególnych elementów na stronie, takich jak nagłówki, sekcje i paragrafy.

background-color ustawia kolor tła

background-imageustawia obraz tła

                    
body {

  background-color: lightblue;

}
                    
                
                    
body {

  background-image: url(bg-image.jpg);

}

background-position ustawia pozycję obrazu tła

background-size ustawia rozmiar obrazu tła

                    
body {

  background-image: url(bg-image.jpg);

  background-position: center;

}
                    
                
                    
body {

  background-image: url(bg-image.jpg);

  background-size: cover;

}

Krycie I Przejrzystość Tła

background-clip Krycie tła

opacity Przezroczystość tła

                    
body {

  background-color: lightblue;

  background-clip: border-box;

}
                    
                
                    
body {

  background-color: lightblue;

  opacity: 0.8;

}
Uwaga

Należy pamiętać, że przezroczystość tła dotyczy tylko koloru tła, a nie obrazu tła. Aby ustawić przezroczystość obrazu tła, należy go najpierw umieścić w osobnym elemencie HTML i ustawić mu przezroczystość.
