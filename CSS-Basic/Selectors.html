Selektor Elementu

Selektor elementu w CSS to jeden z najprostszych i najczęściej używanych selektorów, który pozwala na wybieranie elementów HTML na podstawie nazwy tagu. Selektor elementu jest tworzony poprzez podanie nazwy tagu bez żadnych dodatkowych symboli. Przykład selektora elementu to:

Selektor Elementu "p" wybierze wszystkie elementy <p> na stronie i nada im kolor niebieski. W efekcie wszystkie teksty zawarte w elemencie "p" będą wyświetlane na niebiesko. 

								
p {

	color: blue;

Selektor Klasy

Selektor klasy pozwala na wybieranie elementów HTML o określonej klasie. Klasa jest atrybutem HTML, który można nadać dowolnemu elementowi i służy do grupowania elementów w celu łatwiejszego stylowania. Selektor klasy jest identyfikowany kropką przed nazwą klasy .nazwaklasy.

W poniższym przykładzie selektor klasy ".example" wybierze wszystkie elementy z klasą "example" i nada im kolor niebieski oraz rozmiar czcionki 16px. Aby użyć tego selektora klasy, należy nadać atrybut "class" odpowiedniemu elementowi HTML: 

								
.example {

	color: blue;

	font-size: 16px;

}
						
										

<p class="example">Tekst z selektorem klasy</p>

Selektor ID

Selektor ID to jeden z selektorów, który pozwala na wybieranie pojedynczego elementu HTML na podstawie jego unikalnego identyfikatora (atrybutu id). Selektor ID jest tworzony poprzez podanie symbolu # i nazwy identyfikatora bez cudzysłowu.

W poniższym przykładzie selektor ID "#example" wybierze element o ID "example" i nada mu kolor niebieski oraz pogrubioną czcionkę. Aby użyć tego selektora ID, należy nadać atrybut "id" odpowiedniemu elementowi HTML. 

								
#example {

	color: blue;

	font-weight: bold;

}
						
										

<p id="example">Tekst z selektorem ID</p>

Selektor Potomstwa

Selektor potomstwa pozwala na wybieranie elementów HTML, które są potomkami innego elementu. Selektor potomka jest tworzony poprzez podanie kilku selektorów połączonych ze sobą bezpośrednio.

W poniższym przykładzie selektor potomka "div p" wybierze wszystkie elementy <p> , które są bezpośrednimi potomkami elementów <div>. W efekcie wszystkie teksty zawarte w elemencie, <p> są bezpośrednim potomkiem <div> , będzie miał kolor niebieski i kursywę. 

								
div p {

	color: blue;

	font-style: italic;

}
						
										
<div>

	<p>Tekst z selektorem potomka</p>

</div>

Selektor Rodzeństwa

Selektor rodzeństwa pozwala na wybieranie elementów HTML, które są kolejnymi rodzeństwami innego elementu. Selektor rodzeństwa jest tworzony przez podanie dwóch selektorów oddzielonych znakiem plusa +.

W poniższym przykładzie selektor kolejnego rodzeństwa "h3 + p" wybierze wszystkie elementy <p> , które są kolejnymi rodzeństwami elementów <h3> . W efekcie wszystkie teksty zawarte w elemencie <p> są rodzeństwem elementu <h3>, będą miały kolor niebieski i pogrubioną czcionkę. 

								
h3 + p {

	color: blue;

	font-weight: bold;

}
						
										
<h1>Nagłówek</h1>

<p>
		Tekst z selektorem rodzeństwa
</p>
