Wysokość I Szerokość CSS

W CSS, szerokość i wysokość elementu można ustawić za pomocą właściwości "width" i "height". Te właściwości określają odpowiednio szerokość i wysokość elementu w pikselach lub jednostkach procentowych w stosunku do rodzica.

Wartości width i height można określić w różnych jednostkach, takich:

  • px
  • %
  • cm
  • in
  • em

widthOkreśla szerokość elementu.

heightOkreśla wysokość elementu.

                    
.element {

  width: 500px;

}
                    
                
                    
.element {

  height: 300px;

}

min-widthOkreśla minimalną szerokość elementu.

max-widthOkreśla maksymalną szerokość elementu.

                    
.element {

  min-width: 200px;

}
                    
                
                    
.element {

  max-width: 800px;

}

min-heightOkreśla minimalną wysokość elementu.

max-heightOkreśla maksymalną wysokość elementu.

                    
.element {

  min-height: 100px;

}
                    
                
                    
.element {

  max-height: 500px;

}

  Ten element ma 100% szerokości oraz 86px wysokości
