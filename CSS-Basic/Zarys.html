Outline - Zarys CSS

Właściwość CSS "outline" jest używana do tworzenia obrysu wokół elementu HTML, który jest niezależny od granic (border) i odstępów (margin). Outline jest widoczny na stronie i może być użyty do wizualnego zaznaczenia elementów na stronie, np. podczas debugowania i testowania.

Można ustawić styl outline poprzez ustawienie właściwości CSS: outline-style outline-color outline-width 

                    
element {
    outline: hotpink solid 12px;
    width: 350px;
    border: 4px solid white;
    padding: 20px;
    margin: 60px;
}
Ten element ma zieloną obwódkę i różowy kontur o szerokości 12px.

Wszystkie Style Outline W CSS

dotted Obrys złożony z kropek.

dashed Obrys złożony z kresek.

                    
element {
  outline-width: 2px;
  outline-style: dotted;
  outline-color: blue;
}
                    
                
                    
element {
  outline-width: 2px;
  outline-style: dashed;
  outline-color: blue;
}

Solid Obrys w pełni wypełniony kolorem.

double Podwójny obrys z paskiem

                    
element {
  outline-width: 2px;
  outline-style: solid;
  outline-color: blue;
}
                    
                
                    
element {
  outline-width: 2px;
  outline-style: double;
  outline-color: blue;
}

groove Obrys w stylu "rowka".

ridge Obrys w stylu "grzbietu".

                    
element {
  outline-width: 2px;
  outline-style: groove;
  outline-color: blue;
}
                    
                
                    
element {
  outline-width: 2px;
  outline-style: ridge;
  outline-color: blue;
}

inset Wnętrze obrysu jest wcięte.

outset Wnętrze obrysu jest wypukłe.

                    
element {
  outline-width: 2px;
  outline-style: inset;
  outline-color: blue;
}
                    
                
                    
element {
  outline-width: 2px;
  outline-style: outset;
  outline-color: blue;
}

Uwaga

Obrys jest inny niż ramka (border) - Obrys nie zajmuje miejsca na stronie i nie wpływa na rozmiar elementu, podczas gdy ramka jest widoczna i zajmuje miejsce na stronie.
