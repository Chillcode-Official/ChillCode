Marginesy W Języku CSS

Marginesy to przestrzenie wokół elementów na stronie internetowej, które oddzielają je od innych elementów i pozwalają na lepsze formatowanie strony. Można je ustawić za pomocą CSS, używając właściwości "margin-top", "margin-right", "margin-bottom" i "margin-left". Wartość marginesów może być ustawiona w jednostkach pikseli, procentach, punktach, milimetrach itp.

Margin-top

Margin-right

                    
.element {
  margin-top: 30px;
}
                    
                
                    
.element {
  margin-right: 10%;
}
Ten element ma margines top 30px.

Margin-bottom

Margin-left

                    
.element {
  margin-bottom: 30px;
}
                    
                
                    
.element {
  margin-left: 80px;
}
Ten element ma margines left 80px.

Margin jest to skrócona wersja wszystkich właściwości marginesów.

Można też ustawić jedną wartość dla wszystkich marginesów wtedy wszystkie marginesy będą miały tę samą wartość 

                
.element {
  margin: 10px 20px 30px 40px;
}

-------------------------------------------------

.element {
  margin: 10px;
}
Uwaga

Marginesy w CSS są jednym z najważniejszych elementów projektowania stron internetowych. Dają one projektantom kontrolę nad tym, jak elementy na stronie będą wyglądać i jak będą ze sobą współdziałać.
