Kolory W Języku CSS

CSS umożliwia definiowanie kolorów dla różnych elementów na stronie, np. tekstu, tła, obramowania itp. Kolory w CSS są definiowane za pomocą właściwości "color" lub "background-color". Kolory w CSS mogą być określane na kilka sposobów, w tym:

Kolory nazwane

Kolory RGB

                    
p {

   color: blue;

}
                    
                
                    
p {

   color: rgb(0, 0, 255);

}

Kolory HEX

Kolory HSL

                    
p {

   color: #0000FF;

}
                    
                
                    
p {

   color: hsl(240, 100%, 50%);

}

Nazwy Kolorów

W HTML kolor można określić za pomocą nazwy koloru. więcej kolorów znajdziesz tu

Indian Red

Blue Violet

Dark Slate Gray

Medium Aquamarine

Slate Blue

Coral

Sandy Brown

Hot Pink

Oto przykład definiowania kolorów w CSS:

                
p {
   color: blue;

   background-color: yellow;
}


h1 {
   color: rgb(255, 0, 0);

   background-color: rgb(0, 255, 0);
}


div {
   color: #FF0000;

   background-color: #00FF00;
}


span {
   color: hsl(0, 100%, 50%);

   background-color: hsl(120, 100%, 50%);
}
