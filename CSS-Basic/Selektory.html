Selektor Elementu CSS

Selektor elementu w CSS to jeden z najprostszych i najczęściej używanych selektorów, który pozwala na wybieranie elementów HTML na podstawie nazwy tagu. Selektor elementu jest tworzony poprzez podanie nazwy tagu bez żadnych dodatkowych symboli. Przykład selektora elementu to:

Selektor Elementu "p" wybierze wszystkie elementy <p> na stronie i nada im kolor niebieski. W efekcie wszystkie teksty zawarte w elemencie <p> będą wyświetlane na niebiesko. 

                
p {

  color: blue;

Selektor Klasy

Selektor klasy w CSS pozwala na wybieranie elementów HTML o określonej klasie. Klasa jest atrybutem HTML, który można nadać dowolnemu elementowi i służy do grupowania elementów w celu łatwiejszego stylowania. Selektor klasy jest identyfikowany przez kropkę (.) przed nazwą klasy.

W poniższym przykładzie selektor klasy ".example" wybierze wszystkie elementy z klasą "example" i nada im kolor niebieski oraz rozmiar czcionki 16px. Aby użyć tego selektora klasy, należy nadać atrybut "class" odpowiedniemu elementowi HTML: 

                
.example {

  color: blue;

  font-size: 16px;

}
            
                    

<p class="example">Tekst z selektorem klasy</p>

W powyższym przykładzie element <p> będzie miał kolor niebieski i rozmiar czcionki 16px, ponieważ ma przypisaną klasę "example".

Selektor ID

Selektor ID w CSS to jeden z selektorów, który pozwala na wybieranie pojedynczego elementu HTML na podstawie jego unikalnego identyfikatora (atrybutu "id"). Selektor ID jest tworzony poprzez podanie symbolu "#" i nazwy identyfikatora bez cudzysłowu.

W poniższym przykładzie selektor ID "#example" wybierze element o ID "example" i nada mu kolor niebieski oraz pogrubioną czcionkę. Aby użyć tego selektora ID, należy nadać atrybut "id" odpowiedniemu elementowi HTML. 

                
#example {

  color: blue;

  font-weight: bold;

}
            
                    

<p id="example">Tekst z selektorem ID</p>

W powyższym przykładzie element <p> będzie miał kolor niebieski i pogrubioną czcionkę, ponieważ ma przypisane ID "example".

Selektor Potomstwa

Selektor potomstwa w CSS pozwala na wybieranie elementów HTML, które są potomkami innego elementu. Selektor potomka jest tworzony poprzez podanie kilku selektorów połączonych ze sobą bezpośrednio, bez spacji. Przykład selektora potomka to:

W poniższym przykładzie selektor potomka "div p" wybierze wszystkie elementy <p>, które są bezpośrednimi potomkami elementów <div>. W efekcie wszystki tekst zawarty w elemencie <p>, który jest bezpośrednim potomkiem elementu <div>, będzie miał kolor niebieski i kursywę. 

                
div p {

  color: blue;

  font-style: italic;

}
            
                    
<div>

  <p>Tekst z selektorem potomka</p>

</div>

W powyższym przykładzie tekst zawarty w elemencie <p>, który jest bezpośrednim potomkiem elementu <div>, będzie miał kolor niebieski i kursywę.

Selektor Rodzeństwa

Selektor rodzeństwa w CSS pozwala na wybieranie elementów HTML, które są kolejnymi rodzeństwami innego elementu. Selektor rodzeństwa jest tworzony przez podanie dwóch selektorów oddzielonych znakiem "+".

W poniższym przykładzie selektor kolejnego rodzeństwa "h1 + p" wybierze wszystkie elementy <p>, które są kolejnymi rodzeństwami elementów <h1>. W efekcie wszystki tekst zawarty w elemencie <p>, który jest kolejnym rodzeństwem elementu <h1>, będzie miał kolor niebieski i pogrubioną czcionkę. 

                
h1 + p {

  color: blue;

  font-weight: bold;

}
            
                    
<h1>Nagłówek</h1>

<p>
    Tekst z selektorem rodzeństwa
</p>

W powyższym przykładzie tekst zawarty w elemencie <p>, który jest kolejnym rodzeństwem elementu <h1>, będzie miał kolor niebieski i pogrubioną czcionkę.
