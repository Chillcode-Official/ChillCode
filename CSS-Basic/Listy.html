Listy W Języku CSS

Listy w CSS to elementy HTML, które pozwalają na prezentację informacji w postaci uporządkowanej lub nieuporządkowanej listy punktów. W CSS można zastosować wiele właściwości, takich jak typ punktora, położenie punktora, odstępy i marginesy, aby dostosować wygląd i prezentację list.

Nieuporządkowana lista Użyj tagu Użyj tagu ul do utworzenia nieuporządkowanej listy 

                    
<ul style="list-style-type: square;">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
  • Item 1
  • Item 2
  • Item 3

Uporządkowana lista Użyj tagu ol do utworzenia uporządkowanej listy 

                    
<ol style="list-style-type: upper-roman;">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ol>
  1. Item 1
  2. Item 2
  3. Item 3

Zagnieżdżona lista Aby utworzyć zagnieżdżoną listę użyj tagu ul lub ol wewnątrz elementu listy. 

                    
<ul>
  <li>Item 1
    <ul>
      <li>Subitem 1</li>
      <li>Subitem 2</li>
    </ul>
  </li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
  • Item 1
    • Subitem 1
    • Subitem 2
  • Item 2
  • Item 3

Właściwości List CSS

Przykład, w którym wykorzystano wiele właściwości dla elementów list: 

                
ul {
  list-style: square inside url("bullet.png");
  padding-left: 40px;
  margin: 20px 0;
}

Wszystkie Właściwości List CSS

list-style-type określa typ punktora dla elementów listy.

list-style-image określa obrazek jako punktor dla elementów listy.

                    
ul {
  list-style-type: square;
}
                    
                
                    
ul {
  list-style-image: url('bullet.png');
}

list-style-position określa położenie punktora względem tekstu elementu listy.

list-style skrócona właściwość, która określa wszystkie jednocześnie. 

                    
ul {
  list-style-position: inside;
}
                    
                
                    
ul {
  list-style: square inside;
}

padding-left określa odstęp z lewej strony dla elementów listy.

margin określa marginesy dla elementów listy.

                    
ul {
  padding-left: 20px;
}
                    
                
                    
ul {
  margin: 20px 0;
}
