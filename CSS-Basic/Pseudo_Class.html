Pseudo Klasy CSS

CSS pseudo Klasy to są specjalne klasy, które umożliwiają stylizowanie elementów na stronie w zależności od ich stanu, takiego jak: aktywność, pozycja w dokumencie itp. Pseudo-klasy wyglądają jak klasy, ale są one poprzedzone znakiem dwukropka :klasa.

:hover prosty przykład użycia

:active prosty przykład użycia

										
.tooltip {
  position: relative;
  display: inline-block;
  border-bottom: 1px dotted black;
}

.tooltip .tooltiptext {
  visibility: hidden;
  width: 200px;
  background-color: black;
  color: #fff;
  text-align: center;
  border-radius: 6px;
  padding: 5px 0;
  position: absolute;
  z-index: 1;
  bottom: 125%;
  left: 50%;
  margin-left: -60px;
  opacity: 0;
  transition: opacity 0.3s;
}

.tooltip:hover .tooltiptext {
  visibility: visible;
  opacity: 1;
}
