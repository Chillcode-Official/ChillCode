Obramowanie

W CSS, border to właściwość, która umożliwia dodanie obramowania wokół elementu HTML. Obramowanie składa się z trzech głównych części: szerokości, stylu i koloru. Szerokość określa grubość linii, styl określa jak linie będą wyglądać (np. ciągłe, kropkowane, grube itp.), a kolor określa kolor linii. Obramowanie można ustawić na jednej, kilku lub wszystkich stronach elementu.

CSS pozwala na bardzo dokładne kontrolowanie wyglądu obramowania. Oto główne aspekty obramowania, które można ustawić: width style color 

										
element {
  border-width: 4px;
  border-style: solid;
  border-color: black;
  border-bottom-style: dotted;
  border-left-color: red;
  border-bottom-color: navajowhite;
}
										
								
										
element {
  border: 2px solid black;
}
Ten element div posiada wiele elementów obramowania
Wyjaśnienie

W powyższym przykładzie ustawiono grubość obramowania na 2 piksele, styl na "solid", a kolor na czarny. Obramowanie górne ma grubość 4 pikseli, obramowanie dolne jest kropkowane, a obramowanie lewe jest czerwone.
