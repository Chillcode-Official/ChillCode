Obramowanie

W CSS, border to właściwość, która umożliwia dodanie obramowania wokół elementu HTML. Obramowanie składa się z trzech głównych części: szerokości, stylu i koloru. Szerokość określa grubość linii, styl określa jak linie będą wyglądać (np. ciągłe, kropkowane, grube itp.), a kolor określa kolor linii. Obramowanie można ustawić na jednej, kilku lub wszystkich stronach elementu.

CSS pozwala na bardzo dokładne kontrolowanie wyglądu obramowania. Oto główne aspekty obramowania, które można ustawić: width style color 

                    
element {
  border-width: 4px;
  border-style: solid;
  border-color: black;
  border-bottom-style: dotted;
  border-left-color: red;
  border-bottom-color: navajowhite;
}
                    
                
                    
element {
  border: 2px solid black;
}
Ten element div posiada wiele elementów obramowania
Wyjaśnienie

W powyższym przykładzie ustawiono grubość obramowania na 2 piksele, styl na "solid", a kolor na czarny. Obramowanie górne ma grubość 4 pikseli, obramowanie dolne jest kropkowane, a obramowanie lewe jest czerwone.

Niektóre Style Obramowania W CSS

Solid Obramowanie jest ciągłe i jednolite.

Dotted Obramowanie składa się z kropek.

                    
element {
  border-width: 2px;
  border-style: solid;
  border-color: black;
}
                    
                
                    
element {
  border-width: 2px;
  border-style: dotted;
  border-color: black;
}

Dashed Obramowanie składa się z kresek.

Double Definiuje podwójną granicę

                    
element {
  border-width: 2px;
  border-style: dashed;
  border-color: black;
}
                    
                
                    
element {
  border-width: 2px;
  border-style: double;
  border-color: black;
}

Ridge Definiuje prążkowane obramowanie

  border-style: groove;
  border-color: black;
}

none Obramowanie nie jest widoczne.

hidden Obramowanie jest ukryte

  border-width: 0;

Note

Obramowanie może zajmować dużo miejsca i wpłynąć na wygląd strony. Dlatego ważne jest, aby dobrze przemyśleć ustawienia obramowania i upewnić się, że nie wprowadzają one negatywnego wpływu na wygląd strony.
