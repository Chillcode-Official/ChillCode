Zarys

Właściwość CSS "outline" jest używana do tworzenia obrysu wokół elementu HTML, który jest niezależny od granic (border) i odstępów (margin). Outline jest widoczny na stronie i może być użyty do wizualnego zaznaczenia elementów na stronie.

Można ustawić styl outline poprzez ustawienie właściwości CSS:
outline-style outline-color outline-width 

										
element {
		outline: hotpink solid 12px;
		width: 350px;
		border: 4px solid green;
		padding: 20px;
		margin: 60px;
}
Ten element ma zieloną obwódkę i różowy kontur o szerokości 12px.

Niektóre Style zarysu W CSS

dotted Obrys złożony z kropek.

dashed Obrys złożony z kresek.

										
element {
	outline-width: 2px;
	outline-style: dotted;
	outline-color: blue;
}
										
								
										
element {
	outline-width: 2px;
	outline-style: dashed;
	outline-color: blue;
}

Note

Obrys jest inny niż ramka (border) - Obrys nie zajmuje miejsca na stronie i nie wpływa na rozmiar elementu, podczas gdy ramka jest widoczna i zajmuje miejsce na stronie.
