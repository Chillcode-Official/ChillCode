Niektóre Style zarysu W CSS

dotted Obrys złożony z kropek.

dashed Obrys złożony z kresek.

										
element {
  outline-width: 2px;
  outline-style: dotted;
  outline-color: blue;
}
										
								
										
element {
  outline-width: 2px;
  outline-style: dashed;
  outline-color: blue;
}

Dotted

Dashed

Solid Obrys w pełni wypełniony kolorem.

double Podwójny obrys z paskiem

										
element {
  outline-width: 2px;
  outline-style: solid;
  outline-color: blue;
}
										
								
										
element {
  outline-width: 2px;
  outline-style: double;
  outline-color: blue;
}

Solid

Double

groove Obrys w stylu "rowka".

ridge Obrys w stylu "grzbietu".

										
element {
  outline-width: 2px;
  outline-style: groove;
  outline-color: blue;
}
										
								
										
element {
  outline-width: 2px;
  outline-style: ridge;
  outline-color: blue;
}

Groove

Ridge

inset Wnętrze obrysu jest wcięte.

outset Wnętrze obrysu jest wypukłe.

										
element {
  outline-width: 2px;
  outline-style: inset;
  outline-color: blue;
}
										
								
										
element {
  outline-width: 2px;
  outline-style: outset;
  outline-color: blue;
}

Inset

Outset

Note

Obrys jest inny niż ramka (border) - Obrys nie zajmuje miejsca na stronie i nie wpływa na rozmiar elementu, podczas gdy ramka jest widoczna i zajmuje miejsce na stronie.
