Selektor ID

Selektor ID to jeden z selektorów, który pozwala na wybieranie pojedynczego elementu HTML na podstawie jego unikalnego identyfikatora (atrybutu id). Selektor ID jest tworzony poprzez podanie symbolu # i nazwy identyfikatora bez cudzysłowu.

W poniższym przykładzie selektor ID "#example" wybierze element o ID "example" i nada mu kolor niebieski oraz pogrubioną czcionkę. Aby użyć tego selektora ID, należy nadać atrybut "id" odpowiedniemu elementowi HTML. 

								
#example {

  color: blue;

  font-weight: bold;

}
						
										

<p id="example">Tekst z selektorem ID</p>
