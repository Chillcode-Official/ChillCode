Tabele W Języku CSS

CSS pozwala na zmianę wyglądu tabeli, takiego jak kolory, kształty, marginesy i wiele innych. To pozwala na uzyskanie bardziej atrakcyjnego wyglądu tabel i umożliwia łatwą zmianę wyglądu w całym dokumencie bez potrzeby modyfikowania każdej tabeli oddzielnie.

Oto przykład, jak można wykorzystać CSS do formatowania tabeli: 

                    
table {
    width: 75%;
    margin: 0 auto;
    border-collapse: collapse;
}

th {
    border: 2px solid #d6aeff;
    padding: 8px;
    background-color: #8c8686;
    font-weight: bold;
}

td{
    background-color: #696666;
}

Właściwości Tabeli CSS

border - określa grubość obramowania

width i height - szerokość i wysokość

                    
td, th {
  border: 2px solid black;
}
                    
                
                    
td, th {
  width: 150px;
  height: 50px;
}

text-align - określa wyrównanie tekstu w komórkach tabeli

vertical-align - określa wyrównanie pionowe zawartości komórek tabeli

                    
td, th {
  text-align: center;
}
                    
                
                    
td, th {
  vertical-align: top;
}

background-color - określa kolor tła tabeli

padding - określa odstęp od komórek tabeli

                    
td, th {
  background-color: lightgray;
}
                    
                
                    
td, th {
  padding: 10px;
}
Note

Pamiętaj, że te reguły dotyczą tylko wybranych komórek w tabeli. Aby dostosować wygląd całej tabeli, reguły te należy dostosować do table zamiast do td i th
