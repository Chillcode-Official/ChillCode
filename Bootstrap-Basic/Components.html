Komponenty

Bootstrap 5 to framework front-endowy, który oferuje wiele gotowych komponentów, które można wykorzystać do tworzenia responsywnych i atrakcyjnych stron internetowych. Komponenty w Bootstrap 5 to gotowe elementy interfejsu użytkownika, takie jak menu nawigacyjne, przyciski, formularze, karty produktów, okna modalne, sliderów i wiele więcej.

Button

Button - przycisk

class="btn btn-primary">Przycisk</button>

Navbar - nawigacja strony internetowej 

												
<nav class="navbar navbar-expand-lg navbar-light bg-light">
	<div class="container-fluid">
		<a class="navbar-brand" href="#">Moja strona</a>
		<button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarNav" aria-controls="navbarNav" aria-expanded="false" aria-label="Toggle navigation">
			<span class="navbar-toggler-icon"></span>
		</button>
		<div class="collapse navbar-collapse" id="navbarNav">
			<ul class="navbar-nav">
				<li class="nav-item">
					<a class="nav-link active" aria-current="page" href="#">Strona główna</a>
				</li>
				<li class="nav-item">
					<a class="nav-link" href="#">O nas</a>
				</li>
				<li class="nav-item">
					<a class="nav-link" href="#">Kontakt</a>
				</li>
			</ul>
		</div>
	</div>
</nav>
