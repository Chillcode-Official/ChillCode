Kolory W Jezyku HTML

W języku HTML, kolory mogą być określane za pomocą kodów szesnastkowych (np. #FF0000 dla czerwonego) lub nazw (np. "red" dla czerwonego). Można je używać w atrybutach HTML, takich jak "style" lub "bgcolor" do określenia kolorów tekstu, tła itp.

Aby ustawić kolor tła elementu HTML, można użyć atrybutu style lub bgcolor 

                
<h1>Hello world<h1>

<body style="background-color: #FFC0CBFF;">

<body bgcolor="pink">

Aby ustawić kolor tekstu, można użyć atrybutu style lub color 

                
<p style="color: blue;">Tekst</p>

<font color="green">Tekst</font>

Wartości Kolorów

W języku HTML i CSS istnieją 4 różne systemy wartości kolorów: RGB (czerwień, zieleń, niebieski), HEX (heksadecymalne), HSL (odcień, nasycenie, jasność) i CMYK (cyjan, magenta, żółty, key). Każdy z nich określa kolor przez różne składniki i jest używany w różnych sytuacjach, np. RGB jest często używany w tworzeniu stron internetowych, podczas gdy CMYK jest używany głównie w druku.

Ustawienie koloru tła strony internetowej przy użyciu rgb 

                
<body style="background-color: rgb(128, 255, 0);">

Ustawienie koloru tła strony internetowej przy użyciu hsla 

                
<body style="background-color: hsla(9, 100%, 64%, 0.6);">
