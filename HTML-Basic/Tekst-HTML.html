Formatowanie Tekstu W HTML

Formatowanie tekstu w HTML polega na umieszczaniu znaczników wokół poszczególnych fragmentów tekstu, aby określić ich wygląd i formatowanie. Poniżej przedstawiono kilka przykładowych znaczników HTML do formatowania tekstu:

Poniżej przedstawiono wszystkie znaczniki HTML do formatowania tekstu:

Znacznik <p> 

                
<p>To jest przykładowy akapit tekstu. Może zawierać kilka zdań.</p>

Znacznik <h1> do <h6> 

                
<h1>Tytuł artykułu</h1>

Znacznik <strong> 

                
<p>To jest przykład <strong>pogrubionego</strong> tekstu.</p>

Znacznik <em> 

                
<p>To jest przykład <em>kursywionego</em> tekstu.</p>

Znacznik <u> 

                
<p>To jest przykład <u>podkreślonego</u> tekstu.</p>

Znacznik <br> 

                
<p>To jest przykład tekstu z <br>znakiem nowej linii.</p>

Znacznik <hr> 

                
<hr>

Znacznik <sup> 

                
<p>To jest przykład tekstu z <sup>indeksem górnym</sup></p>

Znacznik <sub> 

                
<p>To jest przykład tekstu z <sub>indeksem dolnym</sub></p>

Znacznik <i> 

                
<p>To jest przykład <i>pochylonego</i> tekstu.</p>

Znacznik <small> 

                
<small>To jest przykład tekstu zmniejszonego</small>

Znacznik <mark> 

                
<mark>To jest przykład tekstu oznaczonego</mark>

Znacznik <s> 

                
<p>To jest przykład <s>przekreślonego</s> tekstu.</p>

Znacznik <ins> 

                
<ins>To jest przykład tekstu wstawionego</ins>

Znacznik <del> 

                
<del>To jest przykład tekstu usuniętego</del>
