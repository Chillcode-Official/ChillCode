Kolory W Jezyku HTML

W języku HTML, kolory mogą być określane za pomocą kodów szesnastkowych (np. #FF0000 dla czerwonego) lub nazw (np. "red" dla czerwonego). Można je używać w atrybutach HTML, takich jak "style" lub "bgcolor" do określenia kolorów tekstu, tła itp.

Aby ustawić kolor tła elementu HTML, można użyć atrybutu style lub bgcolor 

								
<h1>Snorlax <img src="snorlax.png"> <h1>

<body style="background-color: #FFC0CBFF;">

<body bgcolor="pink">

Snorlax

Aby ustawić kolor tekstu, można użyć atrybutu style lub color 

								
<p style="color: blue;">Niebieski rekin <img src="shark.png"> </p>

<font color="green">Jestem porannym ptaszkiem <img src="goblin.png"> </font>

Niebieski rekin

Jestem porannym ptaszkiem
Note

Jest tez mnóstwo bibliotek i narzędzi do generowania kolorów, takich jak "paleta kolorów" czy "picker kolorów" które mogą być pomocne przy tworzeniu stron internetowych.

Kolor Ramki

Ustawienie koloru ramki przy użyciu atrybutu style 

								
<h2 style="border:2px solid Violet;">Hello World</h1>

Hello World

Wartości Kolorów

W języku HTML i CSS istnieją 3 różne systemy wartości kolorów: RGB (czerwień, zieleń, niebieski), HEX (heksadecymalne) i HSL (odcień, nasycenie, jasność). Każdy z nich określa kolor przez różne składniki i jest używany w różnych sytuacjach.

Ustawienie koloru tła strony internetowej przy użyciu rgb 

								
<body style="background-color: rgb(128, 255, 0);">  <!-- kolor zielony -->

Ustawienie koloru tła strony internetowej przy użyciu hsla 

								
<body style="background-color: hsla(9, 100%, 64%, 0.6);">  <!-- kolor pomarańczowy -->
