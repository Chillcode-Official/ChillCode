Pierwszy Kod HTML

Kod HTML (Hypertext Markup Language) jest językiem znaczników, który służy do tworzenia i strukturyzacji treści stron internetowych. Kod HTML składa się z różnych elementów, które określają strukturę i formatowanie treści na stronie internetowej. Poniżej przedstawiam przykład pierwszego kodu HTML: 

                
<!DOCTYPE html>
<html>
  <head>

    <title>Mój pierwszy kod HTML</title>

  </head>
  <body>

    <h1>Witaj świecie!</h1>

    <p>To jest mój pierwszy kod HTML.</p>

  </body>
</html>

Podstawowe Pojęcia

Początek kodu zaczyna się od linii <!DOCTYPE html>, która określa, że jest to dokument HTML5.

Następnie mamy <html> główny element w każdym dokumencie HTML, który zawiera całą zawartość strony.

<head> jest elementem, który zawiera metainformacje dotyczące strony, takie jak tytuł strony, który pojawia się na zakładce przeglądarki.

<body> jest elementem, który zawiera treść strony, którą widzimy na ekranie.

Na końcu, mamy zamykający </html> aby zakończyć dokument.

Wszystkie elementy HTML są zamknięte za pomocą znacznika końcowego / przed nazwą.
