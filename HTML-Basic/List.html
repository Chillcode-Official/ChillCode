Lista Uporządkowana

Lista uporządkowana jest rodzajem listy, w której elementy są uporządkowane według określonego kryterium, takiego jak chronologia, kolejność alfabetyczna lub liczbowa.

Aby utworzyć listę uporządkowaną w HTML, należy użyć tagu <ol> . Każdy element listy powinien być oznaczony tagiem <li> . 

								
<ol>

	<li>Pierwszy element</li>
	<li>Drugi element</li>
	<li>Trzeci element</li>

</ol>
  1. Pierwszy element
  2. Drugi element
  3. Trzeci element

Listy uporządkowane mogą mieć również różne rodzaje numeracji, takie jak rzymskie cyfry lub litery. Można to ustawić za pomocą atrybutu type . 

								
<ol type="I">

	<li>Pierwszy element</li>
	<li>Drugi element</li>
	<li>Trzeci element</li>

</ol>
  1. Pierwszy element
  2. Drugi element
  3. Trzeci element

Lista Nieuporządkowana

HTML Lista Nieuporządkowana to lista, która składa się z elementów nieuporządkowanych, czyli bez określonej kolejności. Elementy te zazwyczaj wyświetlane są w postaci wypunktowania.

Do utworzenia listy nieuporządkowanej w HTML używa się znacznika <ul> oraz dla każdego elementu listy, znacznika <li> . 

								
<ul>

	<li>Pierwszy element</li>
	<li>Drugi element</li>
	<li>Trzeci element</li>

</ul>
  • Pierwszy element
  • Drugi element
  • Trzeci element

list-style-type pozwala na określenie rodzaju punktu przed każdym elementem nieuporządkowanej listy HTML. Wyróżniamy disc circle square none 

								
<ul style="list-style-type:disc;">
	<li>mleko <img src="milk.png"></li>
	<li>miód <img src="honey.png"></li>
</ul>


<ul style="list-style-type:circle;">
	<li>kawa <img src="coffee.png"></li>
	<li>herbata <img src="tea.png"></li>
</ul>


<ul style="list-style-type:square;">
	<li>Lays <img src="chips.png"></li>
	<li>piwo <img src="beer.png"></li>
</ul>


<ul style="list-style-type:none;">
	<li>wódka <img src="vodka.png"></li>
	<li>pepsi <img src="pepsi.png"></li>
</ul>
  • mleko
  • miód
  • kawa
  • herbata
  • Lays
  • piwo
  • pepsi
  • Whisky

Lista Definicyjna

Lista definicyjna w HTML jest używana do tworzenia list, w których każdy element składa się z pojęcia i jego definicji. Listy definicyjne są często używane do tworzenia słowników, list pojęć i innych typów list, w których istnieje pojęcie i jego definicja.

Elementy listy definicyjnej składają się z trzech głównych tagów: <dl> <dt> <dd> 

								
<dl>
 <dt>HTML`</dt>
 <dd>Język opisu stron internetowych</dd><br>

 <dt>ChillCode</dt>
 <dd>Serwis który nauczy cię programowania</dd>
</dl>
HTML'
Język opisu stron internetowych

ChillCode
Serwis który nauczy cię programowania
Note

Warto zauważyć, że tagi <dt> i <dd> muszą być używane w parze i znajdować się wewnątrz tagu <ddl> W przeciwnym razie lista definicyjna nie będzie poprawnie działać.

Zagnieżdżanie List

Zagnieżdżanie list polega na umieszczeniu jednej listy wewnątrz innej listy. Jest to przydatne, gdy chcemy przedstawić zestawienie informacji w postaci hierarchicznej lub gdy chcemy przedstawić informacje w postaci poziomów zagłębienia.

Aby zagnieżdżać listy, należy po prostu umieścić tagi listy wewnątrz odpowiedniego tagu <li> .

Zagnieżdżona Lista Uporządkowana 
								
<ol>
	<li>Pierwszy punkt</li>
	<li>Drugi punkt
		<ol>
			<li>Punkt 2.1</li>
			<li>Punkt 2.2</li>
		</ol>
	</li>
	<li>Trzeci punkt</li>
</ol>
  1. Pierwszy punkt
  2. Drugi punkt
    1. Punkt 2.1
    2. Punkt 2.2
  3. Trzeci punkt
Zagnieżdżona lista Nieuporządkowana 
								
<ul>
	<li>Piwo <img src="beer.png"></li>
	<li>Wino <img src="wine.png"></li>
	<li>
		Jabłka
		<ul>
			<li>Zielone <img src="greenApple.png"></li>
			<li>Czerwone <img src="redApple.png"></li>
		</ul>
	</li>
</ul>
  • Piwo
  • Wino
  • Jabłka
    • Zielone
    • Czerwone
Note

Zagnieżdżanie list w HTML jest bardzo ważne, ponieważ pozwala na hierarchiczną organizację treści i wyświetlanie ich w sposób uporządkowany i czytelny dla użytkowników. Zagnieżdżenie list pozwala również na tworzenie skomplikowanych układów, które można łatwo modyfikować i aktualizować.
