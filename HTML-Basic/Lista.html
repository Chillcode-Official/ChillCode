Lista Uporządkowana

Lista uporządkowana (ang. ordered list) jest rodzajem listy, w której elementy są uporządkowane według określonego kryterium, takiego jak chronologia, kolejność alfabetyczna lub liczbowa.

Aby utworzyć listę uporządkowaną w HTML, należy użyć tagu <ol> (ang. ordered list). Każdy element listy powinien być oznaczony tagiem <li> (ang. list item) 

                
<ol>

  <li>Pierwszy element</li>
  <li>Drugi element</li>
  <li>Trzeci element</li>

</ol>

Listy uporządkowane mogą mieć również różne rodzaje numeracji, takie jak rzymskie cyfry lub litery. Można to ustawić za pomocą atrybutu type tagu <ol>. 

                
<ol type="I">

  <li>Pierwszy element</li>
  <li>Drugi element</li>
  <li>Trzeci element</li>

</ol>
  1. Pierwszy element
  2. Drugi element
  3. Trzeci element
  1. Pierwszy element
  2. Drugi element
  3. Trzeci element

Lista Nieuporządkowana

Nieuporządkowana lista HTML (ang. unordered list) jest tworzona przy użyciu znacznika <ul> (ang. unordered list). Każdy pojedynczy element listy jest oznaczany przez znacznik <li> (ang. list item). W takiej liście każdy element jest oznaczony punktem (bullet point). 

                
<ul>

  <li>Pierwszy element</li>
  <li>Drugi element</li>
  <li>Trzeci element</li>

</ul>
  • Pierwszy element
  • Drugi element
  • Trzeci element

list-style-type pozwala na określenie rodzaju punktu przed każdym elementem nieuporządkowanej listy HTML. Wyróżniamy disc circle square none 

                
<ul style="list-style-type:disc;">
  <li>Honey</li>
  <li>peanuts</li>
  <li>Milk</li>
</ul>


<ul style="list-style-type:circle;">
  <li>Honey</li>
  <li>peanuts</li>
  <li>Milk</li>
</ul>


<ul style="list-style-type:square;">
  <li>Honey</li>
  <li>peanuts</li>
  <li>Milk</li>
</ul>


<ul style="list-style-type:none;">
  <li>Honey</li>
  <li>peanuts</li>
  <li>Milk</li>
</ul>
  • Honey
  • Peanuts
  • Fruits
  • Coffee
  • Tea
  • Milk
  • Lays
  • Beer
  • Juice
  • Vodka
  • Coca-cola
  • Whisky

Lista Definicyjna

Lista definicyjna HTML (ang. definition list) jest używana do tworzenia list, w których każdy element składa się z pojęcia i jego definicji. Listy definicyjne są często używane do tworzenia słowników, list pojęć i innych typów list, w których istnieje pojęcie i jego definicja.

Elementy listy definicyjnej składają się z trzech głównych tagów: <dl> <dt> <dd> 

                
<dl>
 <dt>Html</dt>
 <dd>Język opisu stron internetowych</dd>

 <dt>SoloLearn</dt>
 <dd>Serwis który nauczy cię programowania</dd>
</dl>
HTML
Język opisu stron internetowych

SoloLearn
Serwis który nauczy cię programowania
Uwaga

Warto zauważyć, że tagi <dt> i <dd> muszą być używane w parze i znajdować się wewnątrz tagu <dl> W przeciwnym razie lista definicyjna nie będzie poprawnie działać.

Zagnieżdżanie List

Zagnieżdżanie list HTML polega na umieszczeniu jednej listy wewnątrz innej listy. Jest to przydatne, gdy chcemy przedstawić zestawienie informacji w postaci hierarchicznej lub gdy chcemy przedstawić informacje w postaci poziomów zagłębienia.

Aby zagnieżdżać listy, należy po prostu umieścić tagi listy wewnątrz odpowiedniego tagu <li>.

Zagnieżdżona Lista Uporządkowana

                
<ol>
  <li>Pierwszy element</li>
  <li>Drugi element
    <ol>
      <li>Podelement</li>
      <li>Podelement</li>
    </ol>
  </li>
  <li>Trzeci element</li>
</ol>
  1. Pierwszy element
  2. Drugi element
    1. Podelement
    2. Podelement
  3. Trzeci element

Zagnieżdżona lista Nieuporządkowana

                
<ul>
  <li>Pierwszy element</li>
  <li>Drugi element
    <ul>
      <li>Podelement</li>
      <li>Podelement</li>
    </ul>
  </li>
  <li>Trzeci element</li>
</ul>
