Manipulacja stylami CSS

Manipulacja stylami CSS w jQuery jest ważnym narzędziem w tworzeniu interaktywnych i responsywnych stron internetowych, które dostosowują się do różnych rozmiarów ekranów i urządzeń. Pozwala to na dostosowanie wyglądu strony do preferencji użytkowników i ułatwienie nawigacji po stronie internetowej.

Na przykład, aby zmienić kolor tła elementu o id="myDiv" na czerwony, można użyć: 

                
$("#myDiv").css("background-color", "red");

Aby zmienić kilka stylów CSS jednocześnie, można przekazać obiekt JavaScript do funkcji css(), jak w przykładzie poniżej: 

                
$("#myDiv").css({
  "background-color": "red",
  "font-size": "20px",
  "color": "white"
});

Na przykład, aby dodać klasę "highlight" do elementu o id="myDiv", można użyć: 

                
$("#myDiv").addClass("highlight");

Istnieją wiele innych funkcji jQuery które pozwalazją na zmianę różnych aspektów wyglądu elementów HTML przy użyciu manipulacji stylami CSS, na przykład: toggleClass() removeAttr() height() width() offset()

Uwaga

Manipulacja stylami CSS jest niezbędna w tworzeniu responsywnych stron internetowych, które dostosowują się do różnych urządzeń i rozmiarów ekranów. jQuery ułatwia tę pracę dzięki swoim potężnym funkcjom manipulacji stylami CSS.

Oto kilka przykładów wykorzystania manipulacji stylami CSS w jQuery:

Zmiana koloru tła elementu po kliknięciu: 
                
$("#myDiv").click(function() {
  $(this).css("background-color", "red");
});
Zmiana koloru tekstu elementu po najechaniu kursorem: 
                
$("#myDiv").hover(function() {
  $(this).css("color", "blue");
}, function() {
  $(this).css("color", "black");
});
Zmiana rozmiaru czcionki elementu po kliknięciu: 
                
$("#myDiv").click(function() {
  $(this).css("font-size", "24px");
});
Dodanie klasy CSS do elementu po kliknięciu: 
                
$("#myDiv").click(function() {
  $(this).addClass("highlight");
});
Animowanie zmiany wysokości elementu po kliknięciu: 
                
$("#myDiv").click(function() {
  $(this).animate({
    height: "500px"
  }, 1000);
});
Ukrycie elementu po kliknięciu: 
                
$("#myDiv").click(function() {
  $(this).hide();
});
Zmiana stylów CSS w zależności od rozmiaru okna przeglądarki: 
                
$(window).resize(function() {
  if ($(window).width() < 768) {
    $("#myMenu").addClass("collapsed");
  } else {
    $("#myMenu").removeClass("collapsed");
  }
});
Zmiana tła elementu na losową wartość po każdym odświeżeniu strony: 
                
var colors = ["red", "blue", "green", "yellow", "orange"];

$("#myDiv").css("background-color", colors[Math.floor(Math.random()*colors.length)]);
Uwaga

Te przykłady pokazują tylko kilka sposobów wykorzystania manipulacji stylami CSS w jQuery. Istnieją oczywiście wiele innych możliwości, a jedynym ograniczeniem jest wyobraźnia i potrzeby projektu.
