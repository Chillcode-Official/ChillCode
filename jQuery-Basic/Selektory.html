Selektory jQuery

Selektory w jQuery to ciągi znaków, które pozwalają na identyfikację elementów HTML w drzewie DOM, są bardzo podobne do selektorów CSS i pozwalają na wybieranie elementów na podstawie ich nazw, klas, atrybutów, relacji z innymi elementami, a nawet na podstawie ich pozycji w drzewie DOM. Oto kilka przykładów selektorów w jQuery:

Wybierz wszystkie elementy <p> na stronie:

Wybierz elementy o klasie intro

                    
$("p")
                    
                
                    
$(".intro")

Wybierz elementy o atrybucie href zaczynającym się od https

Wybierz elementy <a> znajdujące się w elemencie <div> o id "main"

                    
$("[href^='https']")
                    
                
                    
$("#main a")

Wybierz elementy <input> typu text znajdujące się w formularzu o id "myForm"

Wybierz elementy <li> znajdujące się w elemencie o klasie "menu" i mające dzieci elementu <a>

                    
$("#myForm input[type='text']")
                    
                
                    
$(".menu li:has(a)")

W jQuery selektory można łączyć ze sobą, aby uzyskać bardziej skomplikowane zapytania. Przykładowo, aby wybrać wszystkie elementy <li> znajdujące się w elemencie <ul> o klasie "menu" oraz mające atrybut "data-price" większy niż 10, można użyć następującego zapytania: 

                 
$("ul.menu li[data-price > '10']")
Wyjaśnienie

Powyższe zapytanie wybierze wszystkie elementy <li> spełniające warunki zapytania, czyli będące dziećmi elementu <ulgt o klasie "menu" oraz posiadające atrybut "data-price" większy niż 10.

jQuery posiada wiele selektorów wbudowanych, ale również można tworzyć własne selektory. Do tego celu można wykorzystać metodę $.expr.createSelector(). Przykładowo, aby stworzyć własny selektor wybierający elementy <a> z atrybutem "target" o wartości "_blank", można użyć następującego kodu: 

                
$.expr.createSelector( "targetBlankA", function( elem ) {

  return $( elem ).is( "a" ) && $( elem ).attr( "target" ) === "_blank";
});
Wyjaśnienie

Powyższy kod stworzy nowy selektor "targetBlankA", który można użyć w zapytaniach tak samo jak selektory wbudowane w jQuery.

Uwaga

Podsumowując, selektory w jQuery to bardzo przydatne narzędzie do wybierania elementów na stronie internetowej. Dzięki nim można w łatwy sposób manipulować elementami HTML i stylami, co przyczynia się do łatwiejszej i bardziej efektywnej pracy z interfejsem użytkownika.

Wiele przykładów użycia selektorów w jQuery

Wybierz elementy <input> typu "checkbox":

Wybierz elementy <input> typu "radio" o wartości "female":

                    
$("input[type='checkbox']")
                    
                
                    
$("input[type='radio'][value='female']")

Wybierz elementy <a> z atrybutem "href" zaczynającym się od "#" lub "mailto:":

Wybierz elementy <option> w elemencie <select> o id "mySelect":

                    
$("a[href^='#'], a[href^='mailto:']")
                    
                
                    
$("#mySelect option")

Wybierz elementy <td> znajdujące się w rzędzie o indeksie 3 w tabeli o klasie "table":

Wybierz elementy <input> typu "text" znajdujące się po elementach <label> z klasą "required":

                    
$(".table tr:eq(3) td")
                    
                
                    
$("label.required + input[type='text']")

Wybierz elementy <div> o klasie "content", które nie są elementami <section> ani <article>:

Wybierz elementy <a> z atrybutem "title" o wartości zaczynającej się od "Open":

                    
$("div.content:not(section, article)")
                    
                
                    
$("a[title^='Open']")

Wybierz elementy <li> znajdujące się w elemencie <ul> o id "myList" oraz posiadające klasę "active":

Wybierz elementy <input> typu "text" znajdujące się w elemencie <form> o id "myForm" oraz posiadające atrybut "name" z wartością "username":

                    
$("#myList li.active")
                    
                
                    
$("#myForm input[type='text'][name='username']")
Uwaga

W jQuery istnieje wiele innych selektorów, które pozwalają na precyzyjne wybieranie elementów na stronie internetowej. Dzięki nim możliwe jest łatwe i szybkie manipulowanie elementami, co przyczynia się do tworzenia interfejsów użytkownika o wysokiej jakości.
